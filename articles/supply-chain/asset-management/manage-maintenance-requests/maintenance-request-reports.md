# <a name="maintenance-request-reports"></a>Rapporter om meldinger

[!include [banner](../../includes/banner.md)]

 

I Aktivastyring kan du generere to rapporter som er knyttet til meldinger. Én rapport viser detaljer, og den andre rapporten inneholder en liste som kan brukes til planlegging og oppfølging.

## <a name="create-a-maintenance-request-details-report"></a>Opprette en rapport om melding

Rapporten **Detaljer om meldinger** viser diverse informasjon som er relatert til meldinger.

1. Velg **Aktivastyring** \> **Rapporter** \> **Meldinger** \> **Detaljer om meldinger**.
2. I hurtigfanen **Poster som skal inkluderes**, kan du velge bestemte meldinger som skal tas med i rapporten.
3. På hurtigfanen **Kjør i bakgrunnen** kan du definere rapportgenerering som en satsvis jobb, etter behov.
4. Velg **OK** for å generere rapporten.

Illustrasjonen nedenfor viser et eksempel på rapporten **Detaljer om meldinger**.

![Rapporten Detaljer om meldinger](media/09-manage-maintenance-requests.png)

## <a name="create-a-maintenance-request-list-report"></a>Opprette en rapport om meldingsliste

Rapporten **Meldingsliste** viser en liste over alle meldinger av samme type.

1. Velg **Aktivastyring** \> **Rapporter** \> **Meldinger** \> **Meldingsliste**.
2. I hurtigfanen **Poster som skal inkluderes**, kan du foreta valg for å definere hvilke meldinger som skal tas med i rapporten.
3. På hurtigfanen **Kjør i bakgrunnen** kan du definere rapportgenerering som en satsvis jobb, etter behov.
4. Velg **OK** for å generere rapporten.

Illustrasjonen nedenfor viser et eksempel på rapporten **Meldingsliste** for alle aktive meldinger.

![Rapporten Meldingsliste](media/10-manage-maintenance-requests.png)

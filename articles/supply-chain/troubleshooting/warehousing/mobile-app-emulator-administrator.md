# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Kan bare bruke administratorrollen for å få tilgang til emulatoren for mobilenhet

## <a name="symptoms"></a>Symptomer

Når du definerer lagre i Supply Chain Management, kan du ikke bruke noen roller unntatt administratorrollen for å få tilgang til emulatoren for mobilenheter.

## <a name="resolution"></a>Løsning

Emulatoren for mobilenhetsappen er satt til bare å fungere med administratorkontoen. For alle test- og aktiveringsprosesser anbefaler vi at du bruker selve mobilappen Lagerstyring.

# <a name="invalid-license-plate-or-location-error-when-scanning-in-the-mobile-app"></a>Feil om ugyldig lisensfeil eller lokasjonsfeil ved skanning i mobilappen

## <a name="symptoms"></a>Symptomer

Når du skanner en nummerskilt-ID eller lokasjon, kan du få følgende feilmelding:

> Nummerskiltet eller lokasjonen er ugyldig.

## <a name="resolution"></a>Løsning

Pass på at nummerskilt-ID-en ikke er reserver av noe annet. Dette problemet oppstod tidligere når verdien som en bruker skannet i mobilappen Lagerstyring, var både en gyldig lokasjon og en gyldig nummerskilt-ID. Dette problemet ble imidlertid løst i versjon 10.0.11.

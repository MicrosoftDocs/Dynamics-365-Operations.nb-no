# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Plukklisterapport viser ikke nye merknader som er knyttet til en journal

KB-nummer: 4611660

## <a name="symptoms"></a>Symptomer

Plukklisterapporten viser ikke nye merknader som er knyttet til en plukklistejournal.

## <a name="resolution"></a>Oppløsning

Plukklisterapporten viser bare merknader som er vedlagt på salgsordrelinjen og salgsordrehodet. Eventuelle nye merknader som er lagt til i fakturajournalen, vises ikke.

# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Brukere kan legge tilbake materiallinjer for produkter som er ferdigmeldt

KB-nummer: 4614721

## <a name="symptoms"></a>Symptomer

Systemet tillater brukere å legge tilbake materiallinjer for produkter som er ferdigmeldt.

## <a name="resolution"></a>Oppløsning

Systemet fungerer som tiltenkt. Råvareforbruk kan justeres helt til produksjonsordren får statusen *Avsluttet*.

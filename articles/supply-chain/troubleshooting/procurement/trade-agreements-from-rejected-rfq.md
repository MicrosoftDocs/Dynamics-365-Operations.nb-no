# <a name="trade-agreements-can-be-created-from-rejected-rfqs"></a>Forretningsavtaler kan opprettes fra avviste tilbudsforespørsler

## <a name="symptoms"></a>Symptomer

Forretningsavtaler kan opprettes fra avviste tilbudsforespørsler. Derfor kan ikke systemet hindre at det opprettes forretningsavtalejournaler hvis tilbudsforespørselslinjen ikke er godtatt.

## <a name="resolution"></a>Løsning

Denne virkemåten er forventet. Du kan opprette forretningsavtaler for et hvilket som helst svar på en tilbudsforespørsel, uansett om de ble godtatt eller avvist. Hvis du vil ha mer informasjon, se [Oversikt over tilbudsforespørsler](/dynamics365/supply-chain/procurement/request-quotations.md).

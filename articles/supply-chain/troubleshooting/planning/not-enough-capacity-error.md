# <a name="not-enough-capacity-could-be-found"></a>Fant ikke nok kapasitet

## <a name="symptoms"></a>Symptomer

Når du kjører planlegging, kan du få følgende feilmelding:

> Kan ikke planlegge produksjonsordren %1. Fant ikke nok kapasitet.

Det er flere årsaker til hvorfor planleggingsmotoren kan mislykkes og utstede denne feilmeldingen. Dette emnet gir informasjon som vil hjelpe deg med å finne roten til feilen og deretter redusere den.

## <a name="resolution"></a>Løsning

Det er flere årsaker til hvorfor planleggingsmotoren kan mislykkes og utstede denne feilmeldingen. For retningslinjer som vil hjelpe deg med å finne rotårsaken til feilen og deretter redusere den, kan du se [Rett opp feil med at planleggingsmotoren ikke finner nok kapasitet](/dynamics365/supply-chain/master-planning/not-enough-capacity-error-resolution).

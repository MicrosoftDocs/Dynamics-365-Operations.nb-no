# <a name="physically-received-purchase-orders-dont-appear-on-the-inventory-closing-report"></a>Fysisk mottatte bestillinger vises ikke i lagerlukkingsrapporten

KB-nummer: 4612595

## <a name="symptoms"></a>Symptomer

Fysisk mottatte bestillinger vises ikke i lagerlukkingsrapporten **Kontrollere åpne antall**.

## <a name="resolution"></a>Oppløsning

Rapporten **Kontroller åpne antall** viser avgangstransaksjoner som ikke kan utlignes mot økonomisk oppdaterte lagermottak per den valgte avslutningsdatoen. Du kan velge å ta med fysiske mottak i rapporten. I så fall vises fysiske mottak hvis de kan dekke avgangstransaksjoner som ikke kan utlignes. For mer informasjon, se [Klargjøre kjøring av lagerlukking](/dynamicsax-2012/appuser-itpro/preparing-to-run-inventory-close).

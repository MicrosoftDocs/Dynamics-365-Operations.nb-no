# <a name="set-up-a-transportation-tender"></a>Definere et transportbetalingsmiddel

[!include [banner](../../includes/banner.md)]

Denne fremgangsmåten viser hvordan du setter opp et transporttilbud. Dette gjøres vanligvis av en transportkoordinator. Du kan bruke prosedyren i USMF-demodatafirmaet.


## <a name="select-a-route"></a>Velg en rute
1. Gå til Transportstyring > Planlegging > Arbeidsområde for lastplanlegging.
2. Fjern merket for Skjul sendt og mottatt.
3. Velg linjen med last-ID 00006.
4. Klikk på Vurdering og ruting.
5. Klikk på Ruter.

## <a name="create-the-transportation-tender"></a>Opprette transporttilbudet
1. Klikk på transporttilbud.
2. Klikk på Ny.
3. Utvid seksjonen Generelt.
4. Angi et tall i feltet Forespurte satser.
5. Klikk på Lagre.
6. Klikk på Oppdater status.
7. Klikk på Send.
8. Velg en rute.

# <a name="weighted-average-with-physical-value-and-marking"></a><span data-ttu-id="09cd6-103">Direkte utligning med fysisk verdi og merking</span><span class="sxs-lookup"><span data-stu-id="09cd6-103">Weighted average with physical value and marking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="09cd6-104">Avveid gjennomsnitt er en lagermodell som er basert på prinsippet for avveid gjennomsnitt, der avganger fra lageret beregnes ut fra gjennomsnittsverdien for varene som mottas i lageret under lagerets periodeavslutning, pluss all lagerbeholdning fra den forrige perioden.</span><span class="sxs-lookup"><span data-stu-id="09cd6-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="09cd6-105">Når du kjører en lageravslutning, utlignes alle kvitteringene mot en virtuell avgang, som holder totalt mottatt antall og verdi.</span><span class="sxs-lookup"><span data-stu-id="09cd6-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="09cd6-106">Denne virtuelle avgangen har et korresponderende virtuelt mottak fra der avgangene utlignes.</span><span class="sxs-lookup"><span data-stu-id="09cd6-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="09cd6-107">På denne måten får alle avgangene samme gjennomsnittskostnad.</span><span class="sxs-lookup"><span data-stu-id="09cd6-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="09cd6-108">Virtuell avgangen og mottak kan vises som en virtuell overføring, kalt lageravslutningsoverføring med avveid gjennomsnitt.</span><span class="sxs-lookup"><span data-stu-id="09cd6-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="09cd6-110">Når du bruker avveid gjennomsnitt, kan du merke lagertransaksjonene slik at et bestemt varemottak utlignes mot en bestemt avgang i stedet for å bruke regelen for avveid gjennomsnitt.</span><span class="sxs-lookup"><span data-stu-id="09cd6-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span> 

<span data-ttu-id="09cd6-111">Vi anbefaler en månedelig lageravslutning når du bruker lagermodellen for avveid gjennomsnitt.</span><span class="sxs-lookup"><span data-stu-id="09cd6-111">We recommend a monthly inventory closing when you use the weighted average inventory model.</span></span>

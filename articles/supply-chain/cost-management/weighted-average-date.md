<span data-ttu-id="52ee3-106">Når du kjører en lageravslutning ved hjelp av dato for avveid gjennomsnitt, utlignes alle daglige mottak mot en virtuell avgang.</span><span class="sxs-lookup"><span data-stu-id="52ee3-106">When you run an inventory closing by using weighted average date, all daily receipts are settled against a virtual issue.</span></span> <span data-ttu-id="52ee3-107">Denne virtuelle avgangen inneholder totalt mottatt antall og verdi for den aktuelle dagen.</span><span class="sxs-lookup"><span data-stu-id="52ee3-107">This virtual issue holds the total received quantity and value for that day.</span></span> <span data-ttu-id="52ee3-108">Den virtuelle frigivelsen har et tilsvarende virtuelt mottak, som frigivelsene utlignes mot.</span><span class="sxs-lookup"><span data-stu-id="52ee3-108">The virtual issue has a corresponding virtual receipt that the issues will be settled against.</span></span> <span data-ttu-id="52ee3-109">Derfor får alle avgangene samme gjennomsnittskostnad.</span><span class="sxs-lookup"><span data-stu-id="52ee3-109">Therefore, all issues receive the same average cost.</span></span> <span data-ttu-id="52ee3-110">Du kan se på denne virtuelle avgangen som en virtuell overføring, som kalles *overføring av avveid gjennomsnitt fra lagerlukking* .</span><span class="sxs-lookup"><span data-stu-id="52ee3-110">The virtual issue and virtual receipt can be considered a virtual transfer that is known as the *weighted average inventory closing transfer* .</span></span>

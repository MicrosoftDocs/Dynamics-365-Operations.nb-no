# <a name="view-cost-object-balance"></a>Vise saldo for kostnadsobjekt

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten. Denne fremgangsmåten er ment for Lagermedarbeideren.

1. Klikk Kostnadsadministrasjon.
2. Klikk Frigitte produkter.
3. Bruk hurtigfilteret til å filtrere på Varenummer-feltet med verdien m0004.
4. Klikk Styr kostnader i handlingsruten.
5. Klikk Kostnadsobjekter.
6. Klikk Fysisk antall.
7. Klikk Dimensjonsvisning.
8. Fjern merket for Lokasjon.
9. Klikk OK.

<span data-ttu-id="9be86-107">Følg disse trinnene for å aktivere behandling av avvik:</span><span class="sxs-lookup"><span data-stu-id="9be86-107">To enable nonconformance management, follow these steps:</span></span>

1.  <span data-ttu-id="9be86-108">Definer lagerstyringsparameterne som er knyttet til avvik:</span><span class="sxs-lookup"><span data-stu-id="9be86-108">Define the Inventory and warehouse management parameters that are related to nonconformances:</span></span>
    -   <span data-ttu-id="9be86-109">Sett alternativet **Bruk kvalitetsstyring** til **Ja**.</span><span class="sxs-lookup"><span data-stu-id="9be86-109">Set the **Use quality management** option to **Yes**.</span></span>
    -   <span data-ttu-id="9be86-110">I **Timepris**-feltet angir du en timepris for arbeid i lokal valuta.</span><span class="sxs-lookup"><span data-stu-id="9be86-110">In the **Hourly rate** field, enter an hourly labor rate in the local currency.</span></span> <span data-ttu-id="9be86-111">Timeprisen brukes til å beregne kostnader for operasjoner som er knyttet til et avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-111">The hourly rate is used to calculate costs for operations that are related to a nonconformance.</span></span> <span data-ttu-id="9be86-112">Timeprisen og de beregnede kostnadene gir referanseinformasjon for et avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-112">The hourly rate and calculated costs provide reference information for a nonconformance.</span></span> <span data-ttu-id="9be86-113">De samhandler ikke med annen funksjonalitet.</span><span class="sxs-lookup"><span data-stu-id="9be86-113">They don't interact with other functionality.</span></span>

3.  <span data-ttu-id="9be86-125">Definer problemtypene som skal tilordnes til avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-125">Define the problem types that will be assigned to nonconformances.</span></span> <span data-ttu-id="9be86-126">Bruk **Problemtyper**-siden til å definere en klassifisering av kvalitetsproblemer som oppdages for de ulike avvikstypene.</span><span class="sxs-lookup"><span data-stu-id="9be86-126">Use the **Problem types** page to define a classification of quality problems that are encountered for the various nonconformance types.</span></span> <span data-ttu-id="9be86-127">Du kan definere følgende avvikstyper: **Intern**, **Kunde**, **Leverandør**, **Tjenesteforespørsel**, **Produksjon** og **Koproduktproduksjon**.</span><span class="sxs-lookup"><span data-stu-id="9be86-127">You can set up the following nonconformance types: **Internal**, **Customer**, **Vendor**, **Service request**, **Production**, and **Co-product production**.</span></span> <span data-ttu-id="9be86-128">Bruk **Avvikstyper**-siden til å godkjenne en problemtype som skal brukes i én eller flere avvikstyper.</span><span class="sxs-lookup"><span data-stu-id="9be86-128">Use the **Non conformance types** page to authorize a problem type to be used in one or more nonconformance types.</span></span> <span data-ttu-id="9be86-129">En problemtype som for eksempel er knyttet til en defektkode, kan gjelde for alle avvikstyper, mens derimot en problemtype som er knyttet til kundeklager, kanskje bare gjelder for avvikstypene **Kunde** og **Tjenesteforespørsel**.</span><span class="sxs-lookup"><span data-stu-id="9be86-129">For example, a problem type that is related to a defect code might apply to all nonconformance types, whereas a problem type that is related to customer complaints might apply only to the **Customer** and **Service request** nonconformance types.</span></span>
4.  <span data-ttu-id="9be86-130">Definer karantenesoner for å gi veiledning om hvordan materialer med defekter skal håndteres.</span><span class="sxs-lookup"><span data-stu-id="9be86-130">Define quarantine zones to provide guidance about defective material should be handled.</span></span> <span data-ttu-id="9be86-131">Bruk **Karantenesoner**-siden til å definere soner som kan tilordnes til et avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-131">Use the **Quarantine zones** page to define zones that can be assigned to a nonconformance.</span></span> <span data-ttu-id="9be86-132">Avviksetiketten som skrives ut, viser den tilordnede karantenesonen og informasjon om bruk, for å gi veiledning om hvordan materialer med defekter skal håndteres.</span><span class="sxs-lookup"><span data-stu-id="9be86-132">The printed nonconformance tag will show the assigned quarantine zone and information about usage, to provide guidance about how to handle defective material.</span></span> <span data-ttu-id="9be86-133">Sonene kan tilsvare lagerlokasjoner eller operasjonsressurser.</span><span class="sxs-lookup"><span data-stu-id="9be86-133">The zones might correspond to inventory locations or operations resources.</span></span>

6.  <span data-ttu-id="9be86-138">Definer de relaterte operasjonene som skal tilordnes til avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-138">Define the related operations that will be assigned to nonconformances.</span></span> <span data-ttu-id="9be86-139">Bruk **Operasjoner**-siden til å definere en klassifisering av arbeidet som kan utføres for et godkjent avvik.</span><span class="sxs-lookup"><span data-stu-id="9be86-139">Use the **Operations** page to define a classification of the work that can be performed for an approved nonconformance.</span></span> <span data-ttu-id="9be86-140">Når du tilordner en relatert operasjon til et avvik, kan du også gi detaljert informasjon, for eksempel informasjon om tilknyttede materialer, arbeidstimer og tillegg som kreves for å utføre operasjonen.</span><span class="sxs-lookup"><span data-stu-id="9be86-140">When you assign a related operation to a nonconformance, you can provide detailed information, such as information about the associated material, labor hours, and miscellaneous charges that are required in order to perform the operation.</span></span> <span data-ttu-id="9be86-141">Denne informasjonen brukes til å beregne en estimert kostnad for operasjonen.</span><span class="sxs-lookup"><span data-stu-id="9be86-141">This information is used to calculate an estimated cost for the operation.</span></span> <span data-ttu-id="9be86-142">Den detaljerte informasjonen og de estimerte kostnadene er til referanse.</span><span class="sxs-lookup"><span data-stu-id="9be86-142">The detailed information and estimated costs are for reference.</span></span> <span data-ttu-id="9be86-143">De relaterte operasjonene for kvalitet er annerledes enn operasjonene som kan defineres for en produksjonsrute.</span><span class="sxs-lookup"><span data-stu-id="9be86-143">The related operations for quality differ from the operations that can be defined for a production route.</span></span>

## <a name="create-a-default-lifecycle-state"></a><span data-ttu-id="3f6c2-105">Opprette en standard livssyklustilstand</span><span class="sxs-lookup"><span data-stu-id="3f6c2-105">Create a default lifecycle state</span></span>

3. <span data-ttu-id="3f6c2-108">Skriv inn en verdi i feltet Tilstand.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-108">In the State field, type a value.</span></span>

4. <span data-ttu-id="3f6c2-118">Skriv inn en verdi i feltet Produktnummer.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-118">In the Product number field, type a value.</span></span>
5. <span data-ttu-id="3f6c2-119">Skriv inn en verdi i feltet Produktnavn.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-119">In the Product name field, type a value.</span></span>

9. <span data-ttu-id="3f6c2-123">Angi eller velg en verdi i lagringsdimensjon-feltet.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-123">In the Storage dimension group field, enter or select a value.</span></span>
10. <span data-ttu-id="3f6c2-124">Angi eller velg en verdi i sporingsdimensjon-feltet.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-124">In the Tracking dimension group field, enter or select a value.</span></span>
11. <span data-ttu-id="3f6c2-125">Klikk på OK.</span><span class="sxs-lookup"><span data-stu-id="3f6c2-125">Click OK.</span></span>

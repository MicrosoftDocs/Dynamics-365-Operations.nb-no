[!include [banner](../../includes/banner.md)]

<span data-ttu-id="95c25-104">Dette emnet viser hvordan du oppretter en strekkode manuelt ved hjelp av varenummer M0001 som eksempel.</span><span class="sxs-lookup"><span data-stu-id="95c25-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="95c25-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="95c25-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="95c25-106">Gå til **Behandling av produktinformasjon \> Produkter \> Frigitte produkter**.</span><span class="sxs-lookup"><span data-stu-id="95c25-106">Go to **Product information management \> Products \> Released products**.</span></span>
1. <span data-ttu-id="95c25-107">Finn og velg ønsket post i listen.</span><span class="sxs-lookup"><span data-stu-id="95c25-107">In the list, find and select the desired record.</span></span> <span data-ttu-id="95c25-108">I dette eksemplet velger du varenummer **M0001**.</span><span class="sxs-lookup"><span data-stu-id="95c25-108">For this example, select item number **M0001**.</span></span>
1. <span data-ttu-id="95c25-109">Velg **Administrer lager** på handlingsruten.</span><span class="sxs-lookup"><span data-stu-id="95c25-109">On the Action Pane, select **Manage inventory**.</span></span>
1. <span data-ttu-id="95c25-110">Velg **Strekkoder**.</span><span class="sxs-lookup"><span data-stu-id="95c25-110">Select **Bar codes**.</span></span>
1. <span data-ttu-id="95c25-111">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="95c25-111">Select **New**.</span></span>
1. <span data-ttu-id="95c25-112">Merk den valgte raden som opprettes i listen nedenfor.</span><span class="sxs-lookup"><span data-stu-id="95c25-112">Mark the selected row that is created in the list below.</span></span>

1. <span data-ttu-id="95c25-115">Skriv inn en verdi i feltet **Strekkode**.</span><span class="sxs-lookup"><span data-stu-id="95c25-115">In the **Bar code** field, type a value.</span></span>  
1. <span data-ttu-id="95c25-116">Angi et tall i **Antall**-feltet.</span><span class="sxs-lookup"><span data-stu-id="95c25-116">In the **Quantity** field, enter a number.</span></span>
1. <span data-ttu-id="95c25-117">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="95c25-117">Select **Save**.</span></span>
1. <span data-ttu-id="95c25-118">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="95c25-118">Close the page.</span></span>

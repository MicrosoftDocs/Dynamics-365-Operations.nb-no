# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="385a2-103">Opprette en strekkode for et produkt</span><span class="sxs-lookup"><span data-stu-id="385a2-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="385a2-104">Dette emnet viser hvordan du oppretter en strekkode manuelt ved hjelp av varenummer M0001 som eksempel.</span><span class="sxs-lookup"><span data-stu-id="385a2-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="385a2-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="385a2-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="385a2-106">Velg **Vedlikehold av frigitt produkt** på startsiden.</span><span class="sxs-lookup"><span data-stu-id="385a2-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="385a2-107">Gå til **Produkter > Frigitte produkter** under **Koblinger**-delen.</span><span class="sxs-lookup"><span data-stu-id="385a2-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="385a2-108">Finn og velg ønsket post i listen.</span><span class="sxs-lookup"><span data-stu-id="385a2-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="385a2-109">I dette eksemplet velger du varenummer **M0001**.</span><span class="sxs-lookup"><span data-stu-id="385a2-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="385a2-110">Velg **Administrer lager** på handlingsruten.</span><span class="sxs-lookup"><span data-stu-id="385a2-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="385a2-111">Velg **Strekkoder**.</span><span class="sxs-lookup"><span data-stu-id="385a2-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="385a2-112">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="385a2-112">Select **New**.</span></span>
7. <span data-ttu-id="385a2-113">Merk den valgte raden som opprettes i listen nedenfor.</span><span class="sxs-lookup"><span data-stu-id="385a2-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="385a2-114">Angi eller velg en verdi i feltet **Strekkodeoppsett**.</span><span class="sxs-lookup"><span data-stu-id="385a2-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="385a2-115">Angi eller velg en verdi i feltet **Strekkode**.</span><span class="sxs-lookup"><span data-stu-id="385a2-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="385a2-116">Skriv inn en verdi i feltet **Strekkode**.</span><span class="sxs-lookup"><span data-stu-id="385a2-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="385a2-117">Angi et tall i **Antall**-feltet.</span><span class="sxs-lookup"><span data-stu-id="385a2-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="385a2-118">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="385a2-118">Select **Save**.</span></span>
13. <span data-ttu-id="385a2-119">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="385a2-119">Close the page.</span></span>

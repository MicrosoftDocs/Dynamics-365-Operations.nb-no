# <a name="set-up-disposition-codes"></a><span data-ttu-id="81fbe-103">Definer disposisjonskoder</span><span class="sxs-lookup"><span data-stu-id="81fbe-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="81fbe-104">Du kan definere disposisjonskoder for å angi hvordan en vare som returneres av en kunde, skal behandles.</span><span class="sxs-lookup"><span data-stu-id="81fbe-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="81fbe-105">Du kan for eksempel opprette en disposisjonskode med navnet **Reparer og returner** for å angi at den returnerte varen skal repareres og returneres til kunden.</span><span class="sxs-lookup"><span data-stu-id="81fbe-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="81fbe-106">Du finner flere eksempler på disposisjonskoder som er vanlige å bruke for varer som returneres av kunder, under [Angi hvordan du kvitter deg med returnerte varer](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="81fbe-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="81fbe-107">Du kan også definere en årsakskode som forklarer hvorfor en vare ble returnert.</span><span class="sxs-lookup"><span data-stu-id="81fbe-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="81fbe-108">Hvis du vil ha mer informasjon om årsakskoder, kan du se [Definere returårsakskoden](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="81fbe-108">For more information about reason codes, see [Set up return reason code](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="81fbe-109">Klikk **Salg og markedsføring** \> **Oppsett** \> **Salgsordrer** \> **Returer** \> **Disposisjonskoder**.</span><span class="sxs-lookup"><span data-stu-id="81fbe-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="81fbe-110">Klikk **Ny** eller trykk CTRL+N for å opprette en ny disposisjonskode.</span><span class="sxs-lookup"><span data-stu-id="81fbe-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="81fbe-111">Angi et unikt beskrivende navn, velg en handling og angi en beskrivelse for disposisjonskoden.</span><span class="sxs-lookup"><span data-stu-id="81fbe-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="81fbe-112">Hvis du vil knytte eventuelle kundegebyr til denne disposisjonskoden, klikker du **Gebyrer**-knappen for å åpne skjemaet **Definer tillegg**.</span><span class="sxs-lookup"><span data-stu-id="81fbe-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="81fbe-113">Hvis du vil definere eksterne koder som skal samsvare med firmaets egne disposisjonskoder, klikker du knappen **Eksterne koder** for å åpne skjemaet **Eksterne koder**.</span><span class="sxs-lookup"><span data-stu-id="81fbe-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>

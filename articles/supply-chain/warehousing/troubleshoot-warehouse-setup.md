# <a name="troubleshoot-warehouse-setup"></a>Feilsøke lageroppsett

[!include [banner](../includes/banner.md)]

Dette emnet beskriver hvordan du løser vanlige problemer som kan oppstå mens du konfigurerer lagre i Microsoft Dynamics 365 Supply Chain Management.

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>Jeg kan ikke bruke noen rolle unntatt administrator til å få tilgang til emulatoren for mobilenhetsappen.

### <a name="issue-description"></a>Problembeskrivelse

Du kan ikke bruke noen rolle unntatt administratorrollen til å få tilgang til emulatoren for mobilenhetsappen.

### <a name="issue-resolution"></a>Problemløsning

Emulatoren for mobilenhetsappen er satt til bare å fungere med administratorkontoen. For alle test- og aktiveringsprosesser anbefaler vi at du bruker selve lagerappen.

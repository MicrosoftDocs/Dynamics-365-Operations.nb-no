# <a name="create-a-location-profile"></a><span data-ttu-id="9f860-103">Opprette en lokasjonsprofil</span><span class="sxs-lookup"><span data-stu-id="9f860-103">Create a location profile</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="9f860-104">Hver lokasjon på lageret må ha en tilknyttet lokasjonsprofil som beskriver egenskapene til lokasjonen, for eksempel om den tillater kombinerte varer.</span><span class="sxs-lookup"><span data-stu-id="9f860-104">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="9f860-105">I denne fremgangsmåten skal vi opprette en profil for en lokasjon som ikke krever nummerskiltkontroll.</span><span class="sxs-lookup"><span data-stu-id="9f860-105">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="9f860-106">Vi skal aktivere kombinerte varer og kombinerte lagerstatuser, og tillate syklustelling.</span><span class="sxs-lookup"><span data-stu-id="9f860-106">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="9f860-107">Du kan bruke prosedyren i USMF-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="9f860-107">You can use this procedure in the USMF demo data company.</span></span>

1. <span data-ttu-id="9f860-108">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="9f860-108">Click New.</span></span>
2. <span data-ttu-id="9f860-109">Skriv inn en verdi i feltet Profil-ID for lokasjon.</span><span class="sxs-lookup"><span data-stu-id="9f860-109">In the Location profile ID field, type a value.</span></span>
3. <span data-ttu-id="9f860-110">Skriv inn en verdi i Navn-feltet.</span><span class="sxs-lookup"><span data-stu-id="9f860-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="9f860-111">Angi eller velg en verdi i feltet Lokasjonsformat.</span><span class="sxs-lookup"><span data-stu-id="9f860-111">In the Location format field, enter or select a value.</span></span>
5. <span data-ttu-id="9f860-112">Angi eller velg en verdi i feltet Lokasjonstype.</span><span class="sxs-lookup"><span data-stu-id="9f860-112">In the Location type field, enter or select a value.</span></span>
6. <span data-ttu-id="9f860-113">Angi eller velg en verdi i feltet ID for sonebehandlingsprofil.</span><span class="sxs-lookup"><span data-stu-id="9f860-113">In the Dock management profile ID field, enter or select a value.</span></span>
7. <span data-ttu-id="9f860-114">Velg Ja i feltet Tillat kombinerte varer.</span><span class="sxs-lookup"><span data-stu-id="9f860-114">Select Yes in the Allow mixed items field.</span></span>
8. <span data-ttu-id="9f860-115">Velg Ja i feltet Tillat kombinerte lagerstatuser.</span><span class="sxs-lookup"><span data-stu-id="9f860-115">Select Yes in the Allow mixed  inventory statuses field.</span></span>
9. <span data-ttu-id="9f860-116">Velg Ja i feltet Tillat syklustelling.</span><span class="sxs-lookup"><span data-stu-id="9f860-116">Select Yes in the Allow cycle counting field.</span></span>
10. <span data-ttu-id="9f860-117">Klikk Lagre.</span><span class="sxs-lookup"><span data-stu-id="9f860-117">Click Save.</span></span>
11. <span data-ttu-id="9f860-118">Gå til Lagerstyring > Oppsett > Lager > Lokasjonsprofiler.</span><span class="sxs-lookup"><span data-stu-id="9f860-118">Go to Warehouse management > Setup > Warehouse > Location profiles.</span></span>

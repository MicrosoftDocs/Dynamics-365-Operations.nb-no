[!include [banner](../includes/banner.md)]

<span data-ttu-id="a6d9d-104">Fordi Android er et open source-prosjekt, kan alle produsenter av maskinvare for lagerstrekkodeskannere bygge en enhet for å kjøre Android-operativsystemet.</span><span class="sxs-lookup"><span data-stu-id="a6d9d-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="a6d9d-105">En enhet er bare Android-kompatibel hvis den kan kjøre apper som er skrevet for Android-kjøringsmiljøet.</span><span class="sxs-lookup"><span data-stu-id="a6d9d-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="a6d9d-106">En maskinvareleverandøren kan imidlertid endre og opprette overlappinger for Android-versjonen som kjører på maskinvaren.</span><span class="sxs-lookup"><span data-stu-id="a6d9d-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="a6d9d-107">Microsoft har ikke ansvar for å sikre at en mobilapp for strekkodeskanning for Android er kompatibel med produsentens maskinvare for strekkodeskanning og Android-versjonen som kjører på den.</span><span class="sxs-lookup"><span data-stu-id="a6d9d-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="a6d9d-110">Som kunde anbefaler vi at du tester lagermobilappen for skanning på valgt maskinvare før du bestemmer deg for maskinvaren som du vil kjøpe.</span><span class="sxs-lookup"><span data-stu-id="a6d9d-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>

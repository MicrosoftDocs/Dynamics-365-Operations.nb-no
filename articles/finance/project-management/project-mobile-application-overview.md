# <a name="project-mobile-applications-overview"></a>Oversikt over mobilapper for Project

[!include [banner](../includes/banner.md)]

Tre mobilapplikasjoner som er knyttet til prosjekttider, er tilgjengelige for Microsoft Dynamics 365 Project Timesheet, Tidsoppføring for prosjekt og Mine timeregistreringer / Timeregistreringer til gjennomgang (optimalisert for mobil).

## <a name="project-timesheet-mobile-app"></a>Mobilappen Project Timesheet

Dynamics 365 Project Timesheet-mobilappen gjør det mulig for brukere å sende og godkjenne timeregistreringer for prosjekter på mobilenheten. Denne mobilappen viser timeregistreringsfunksjonaliteten som befinner seg i prosjektstyrings- og regnskapsområdet på Dynamics 365 Finance. Den øker brukernes produktivitet og effektivitet, og gjør det mulig å legge inn oppføringer til rett tid og godkjenne timeregistreringer for prosjekt.

## <a name="project-time-entry-workspace"></a>Arbeidsområde for tidsoppføring for prosjekt

I arbeidsområdet for tidsoppføring for prosjekt kan brukere angi og spare tid mot et prosjekt. Brukerne kan imidlertid ikke sende eller godkjenne prosjekttimeregistreringer. Den kan brukes bare for tidsoppføring. Dette mobile arbeidsområdet er ment å brukes sammen med Finance and Operations-mobilappen.

## <a name="my-timesheetstimesheets-for-my-review"></a>Mine timeregistrering / Timeregistreringer til gjennomgang

Mine timeregistreringer (optimalisert for mobil) og Timeregistreringer til gjennomgang (optimalisert for mobil) er en løsning som lar ansatte opprette, sende og godkjenne prosjekttimeregistreringer på en mobilenhet. Ansatte får tilgang til Finance-funksjonalitet via URL-adressen for applikasjonen. Denne løsningen krever ikke Finance and Operations-mobilappen.

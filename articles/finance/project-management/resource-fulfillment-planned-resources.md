# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="b0e69-103">Oppfyllelse av planlagte ressurser</span><span class="sxs-lookup"><span data-stu-id="b0e69-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b0e69-104">En prosjektleder kan planlegge nødvendige ressursroller for et prosjekt.</span><span class="sxs-lookup"><span data-stu-id="b0e69-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="b0e69-105">Ressursbehandleren vil se disse planlagte ressursene som forespørsler på siden **Ressursoppfyllelse** og kan tildele faktiske ressurser.</span><span class="sxs-lookup"><span data-stu-id="b0e69-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="b0e69-106">På siden **Alle prosjekter**, velg **XYZ Oppgrader fase 2**-prosjektet.</span><span class="sxs-lookup"><span data-stu-id="b0e69-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="b0e69-107">Velg **Prosjekt** og velg deretter **Rediger**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="b0e69-108">I kategorien **Prosjektteam og planlegging**, velg **Legg til**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="b0e69-109">I dialogboksen **Legg til roller** velger du rollen **Programvareutvikler**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="b0e69-110">Velg **Opprett**, og lukk deretter prosjektsiden.</span><span class="sxs-lookup"><span data-stu-id="b0e69-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="b0e69-111">På siden **Ressursoppfyllelse**, velg **Programvareutvikler 1** for **XYZ oppgraderingsprosjekt fase 2**-prosjektet.</span><span class="sxs-lookup"><span data-stu-id="b0e69-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="b0e69-112">Velg en medarbeider, og deretter **Tildel**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="b0e69-113">Kontroller at linjen for **Programvareutvikler 1** er fjernet for prosjektet **XYZ-oppgraderingsfase 2**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="b0e69-114">I kategorien **Prosjektteam og planlegging** for **XYZ oppgraderingsfase 2**-prosjektet, bekreft at arbeidstageren du valgte i forrige trinn er lagt til som **Programvareutvikler**.</span><span class="sxs-lookup"><span data-stu-id="b0e69-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>

## <a name="set-up-competencies-certificates"></a><span data-ttu-id="6745a-112">Angi kompetanser: Sertifikater</span><span class="sxs-lookup"><span data-stu-id="6745a-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="6745a-113">På listesiden over **Arbeidstakere**,velg den linjen for den arbeideren som det skal legges til sertifikatinformasjon for.</span><span class="sxs-lookup"><span data-stu-id="6745a-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="6745a-114">På Handling-panelet, på **Ansatt**-kategorien, i **Kompetanse**-gruppen, velg **Sertifikater**.</span><span class="sxs-lookup"><span data-stu-id="6745a-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="6745a-115">Velg **Ny**, og deretter i **Sertifikattype**-feltet, velg **PMP**.</span><span class="sxs-lookup"><span data-stu-id="6745a-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="6745a-116">I feltet **Startdato**, velg **10/01/2015**, og velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="6745a-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="6745a-117">Definer kompetanser: Ferdigheter</span><span class="sxs-lookup"><span data-stu-id="6745a-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="6745a-118">På **Arbeidere**-listesiden må du kontrollere at det fortsatt er merket av for arbeideren som du brukte i forrige fremgangsmåte.</span><span class="sxs-lookup"><span data-stu-id="6745a-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="6745a-119">Deretter i Handling-panelet, på **Ansatt**-kategorien, i **Kompetanse**-gruppen, velg **Ferdigheter**.</span><span class="sxs-lookup"><span data-stu-id="6745a-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="6745a-120">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="6745a-120">Select **New**.</span></span>
3. <span data-ttu-id="6745a-121">I **Ferdighet**-feltet velger du **Prosjektstyring**.</span><span class="sxs-lookup"><span data-stu-id="6745a-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="6745a-122">I **Nivå**-feltet velger du **5 Ekspert**.</span><span class="sxs-lookup"><span data-stu-id="6745a-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="6745a-123">I **Nivådato**-feltet velger du **1-/14/2014**.</span><span class="sxs-lookup"><span data-stu-id="6745a-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="6745a-124">I feltet **År med erfaring** angir du **10**.</span><span class="sxs-lookup"><span data-stu-id="6745a-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="6745a-125">Velg **Lagre**, og lukk deretter siden.</span><span class="sxs-lookup"><span data-stu-id="6745a-125">Select **Save**, and then close the page.</span></span>

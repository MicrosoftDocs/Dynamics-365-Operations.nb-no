# <a name="assign-number-sequences"></a>Tilordne nummerserier

[!include [banner](../includes/banner.md)]

Dette emnet beskriver hvordan du oppretter nummerserier for leie-IDer. Det forklarer også hvordan du oppretter unike IDer som brukes i indeksrevurderingsprosessen.

1. Gå til **Aktivaleie \> Oppsett \> Parametere for aktivaleie**.
2. Velg sidekategorien **Nummerserier**.
3. I sidefeltet velger du **Nummerserier**.
4. Velg nummerserien for **Leie-ID**-referansen. Denne nummerserien vil bli brukt til å generere den unike IDen for hver leieavtale.
5. Velg nummerserien for **Prosess-ID**-referansen. Denne nummerserien vil bli brukt til å generere den unike IDen for hver indeksrevalueringsprosess.
6. Velg nummerserien for **Oppsigelsesforslag-ID**-referansen. Denne nummerserien vil bli brukt til å generere den unike IDen for hvert oppsigelsesforslag.

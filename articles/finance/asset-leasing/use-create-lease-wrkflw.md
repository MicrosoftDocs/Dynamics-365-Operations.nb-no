[!include [banner](../includes/banner.md)]

<span data-ttu-id="8a411-104">Dette emnet forklarer hvordan du bruker arbeidsflyter til å godkjenne leie av aktiva, og hvordan du sporer status og historikk for arbeidsflyter.</span><span class="sxs-lookup"><span data-stu-id="8a411-104">This topic explains how to use workflows to approve asset leases, and how to track the status and history of the workflows.</span></span> <span data-ttu-id="8a411-105">Arbeidsflyter bidrar konsekvens i administrasjon av leiegodkjenninger ved å gi et standardsett med godkjenningstrinn og tilordne bestemte brukere som godkjenner hvert trinn i prosessen.</span><span class="sxs-lookup"><span data-stu-id="8a411-105">Workflows help bring consistency to the management of lease approvals by providing a standard set of approval steps and assigning specific users who approve each step of the process.</span></span> <span data-ttu-id="8a411-106">En godkjenner kan godkjenne en leieavtale, avvise den, be om en endring eller tilordne den til en annen bruker for godkjenning.</span><span class="sxs-lookup"><span data-stu-id="8a411-106">An approver can approve a lease, reject it, request a change to it, or assign it to another user for approval.</span></span> <span data-ttu-id="8a411-107">Arbeidsflyter kan også gi mer innsyn i godkjenningsprosessen ved å la deg spore status og logg.</span><span class="sxs-lookup"><span data-stu-id="8a411-107">Workflows can also bring more visibility into the approval process by letting you track their status and history.</span></span> <span data-ttu-id="8a411-108">I tillegg kan du vise en sentralisert arbeidsliste som viser oppgaver og godkjenninger som er tilordnet bestemte godkjennere.</span><span class="sxs-lookup"><span data-stu-id="8a411-108">Additionally, you can view a centralized worklist that lists the tasks and approvals that are assigned to specific approvers.</span></span>

<span data-ttu-id="8a411-109">Før du bruker denne fremgangsmåten må du kontrollere at det er opprettet minst én arbeidsflyt for leiegodkjenning.</span><span class="sxs-lookup"><span data-stu-id="8a411-109">Before you use this procedure, make sure that at least on lease approval workflow has been created.</span></span> <span data-ttu-id="8a411-110">Hvis det ikke finnes noen arbeidsflyt, oppretter du en.</span><span class="sxs-lookup"><span data-stu-id="8a411-110">If no workflow exists, create one.</span></span> <span data-ttu-id="8a411-111">Hvis du vil ha informasjon om hvordan du konfigurerer en arbeidsflyt, kan du se [Konfigurere arbeidsflyter for godkjenning av leie](set-up-lease-wrkflw.md).</span><span class="sxs-lookup"><span data-stu-id="8a411-111">For information about how to set up a workflow, see [Set up lease approval workflows](set-up-lease-wrkflw.md).</span></span>

1. <span data-ttu-id="8a411-112">Send inn en leieavtale til godkjenning.</span><span class="sxs-lookup"><span data-stu-id="8a411-112">Submit a lease for approval.</span></span> <span data-ttu-id="8a411-113">På siden **Tablådetaljer** for leieavtalen velger du **Arbeidsflyt**, og deretter velger du **Send**.</span><span class="sxs-lookup"><span data-stu-id="8a411-113">On the **Book details** page for the lease, select **Workflow**, and then select **Submit**.</span></span>
2. <span data-ttu-id="8a411-114">I dialogboksen som vises, kan du legge til en kommentar.</span><span class="sxs-lookup"><span data-stu-id="8a411-114">In the dialog box that appears, you can add a comment.</span></span> <span data-ttu-id="8a411-115">Den tilordnede godkjenneren vil se denne merknaden sammen med leieavtalen.</span><span class="sxs-lookup"><span data-stu-id="8a411-115">The designated approver will see this comment together with the lease.</span></span> <span data-ttu-id="8a411-116">Når du er ferdig med å legge inn kommentaren, velger du **Send**.</span><span class="sxs-lookup"><span data-stu-id="8a411-116">When you've finished entering the comment, select **Submit**.</span></span> <span data-ttu-id="8a411-117">Leieavtalen sendes til arbeidsflytsystemet, og godkjenneren mottar den for godkjenning.</span><span class="sxs-lookup"><span data-stu-id="8a411-117">The lease is submitted to the workflow system, and the approver receives it for approval.</span></span>
3. <span data-ttu-id="8a411-118">Hvis du vil vise leieavtalene som er utpekt for godkjenning, kan du gå til **Moduler \> Felles \> Arbeidselementer \> Arbeidselementer som er tilordnet til meg**.</span><span class="sxs-lookup"><span data-stu-id="8a411-118">To view the leases that they are designated for approval, go to **Modules \> Common \> Work items \> Work items assigned to me**.</span></span>
4. <span data-ttu-id="8a411-119">På siden **Arbeidselementer som er tilordnet til meg** velger du **leie-ID**-koblingen for leieavtalen du vil vise.</span><span class="sxs-lookup"><span data-stu-id="8a411-119">On the **Work items assigned to me** page, select the **Lease ID** link for the lease that you want to view.</span></span> <span data-ttu-id="8a411-120">Hvilken side som vises, er avhengig av hvilke leietablåer og leiedetaljer du har tilgang til.</span><span class="sxs-lookup"><span data-stu-id="8a411-120">The page that appears depends on the lease books and lease details that you have access to.</span></span>
5. <span data-ttu-id="8a411-121">Når du har fullført visningen av leieavtalen, velger du **Arbeidsflyt** og bestemmer hvilken handling som skal utføres.</span><span class="sxs-lookup"><span data-stu-id="8a411-121">When you've finish viewing the lease, select **Workflow**, and decide what action should be taken.</span></span> <span data-ttu-id="8a411-122">Alternativene omfatter **Godkjenn**, **Avvist**, **Be om endring**, **Deleger** og **Annullert**.</span><span class="sxs-lookup"><span data-stu-id="8a411-122">The options include **Approve**, **Rejected**, **Request change**, **Delegate**, and **Canceled**.</span></span> <span data-ttu-id="8a411-123">Du kan også velge **Vis logg** for å vise godkjenningsloggen for den valgte leieavtalen.</span><span class="sxs-lookup"><span data-stu-id="8a411-123">You can also select **View history** to view the approval history for the selected lease.</span></span>
6. <span data-ttu-id="8a411-124">Når du har valgt en handling, angir du en kommentar som beskriver handlingen.</span><span class="sxs-lookup"><span data-stu-id="8a411-124">After you've selected an action, enter a comment to describe the action.</span></span> <span data-ttu-id="8a411-125">Når du er ferdig med å angi kommentaren, velger du **Godkjent**-handlingen fra listen.</span><span class="sxs-lookup"><span data-stu-id="8a411-125">When you've finished entering the comment, select the **Approved** action in the list.</span></span>
7. <span data-ttu-id="8a411-126">Hvis du vil vise godkjenningshandlingene, går du tilbake til siden **Leiedetaljer** fra siden **Leiesammendrag**, og deretter velger du **Arbeidsflyt \> Vis logg**.</span><span class="sxs-lookup"><span data-stu-id="8a411-126">To view the approval actions, go back to the **Lease details** page from the **Lease summary** page, and then select **Workflow \> View history**.</span></span>

    <span data-ttu-id="8a411-127">Du kan vise arbeidsflytaktivitetene på siden **Arbeidsflytlogg**.</span><span class="sxs-lookup"><span data-stu-id="8a411-127">You can view the workflow activities on the **Workflow history** page.</span></span> <span data-ttu-id="8a411-128">Denne siden viser arbeidsflyttrinnene som er iverksatt på den bestemte leieavtalen.</span><span class="sxs-lookup"><span data-stu-id="8a411-128">This page shows the workflow steps that have been taken on the specific lease.</span></span> <span data-ttu-id="8a411-129">Du kan også bruke **Arbeidselementer**-feltet til å vise statusen for de tilordnede arbeidselementene.</span><span class="sxs-lookup"><span data-stu-id="8a411-129">You can also use the **Work items** field to view the status of the assigned work items.</span></span>

8. <span data-ttu-id="8a411-130">Hvis du vil stoppe en arbeidsflyt, velger du **Tilbakekalling** på siden **Arbeidsflytlogg**.</span><span class="sxs-lookup"><span data-stu-id="8a411-130">To stop a workflow, on the **Workflow history** page, select **Recall**.</span></span> <span data-ttu-id="8a411-131">I dialogboksen som vises, angir du en kommentar, og deretter velger du **OK**.</span><span class="sxs-lookup"><span data-stu-id="8a411-131">In the dialog box that appears, enter a comment, and then select **OK**.</span></span>
9. <span data-ttu-id="8a411-132">Hvis du vil deaktivere en arbeidsflyt, eller aktivere en arbeidsflyt som er opprettet tidligere, kan du gå til **Aktivaleie \> Oppsett \> Leiearbeidsflyt**.</span><span class="sxs-lookup"><span data-stu-id="8a411-132">To inactivate a workflow, or to activate a workflow that was previously created, go to **Asset leasing \> Setup \> Lease workflow**.</span></span> <span data-ttu-id="8a411-133">Deretter, på siden **Leiearbeidsflyt**, velger du **Arbeidsflyt \> Versjoner**.</span><span class="sxs-lookup"><span data-stu-id="8a411-133">Then, on the **Lease workflow** page, select **Workflow \> Versions**.</span></span> <span data-ttu-id="8a411-134">Hvis du vil gjøre en gjeldende arbeidsflyt inaktiv, velger du den aktive leieavtalen i dialogboksen leieversjon, og deretter velger du **Aktiver**.</span><span class="sxs-lookup"><span data-stu-id="8a411-134">To make a current workflow inactive, select the active lease in the lease version dialog box, and then select **Make inactive**.</span></span> <span data-ttu-id="8a411-135">Hvis du vil at en eksisterende arbeidsflyt skal være aktiv, velger du arbeidsflyten og deretter **Gjør aktiv**.</span><span class="sxs-lookup"><span data-stu-id="8a411-135">To make an existing workflow active, select the workflow, and then select **Make active**.</span></span>

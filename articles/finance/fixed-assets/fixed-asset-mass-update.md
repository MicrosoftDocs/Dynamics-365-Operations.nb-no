# <a name="fixed-asset-mass-update"></a><span data-ttu-id="91701-103">Masseoppdatering av faste anleggsmidler</span><span class="sxs-lookup"><span data-stu-id="91701-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="91701-104">Hvis du bruker tablåer, kan du endre avskrivningskonvensjonene for grupper av anleggsmidler som er en del av det samme tablået.</span><span class="sxs-lookup"><span data-stu-id="91701-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="91701-105">Hvis du for eksempel driver virksomhet i USA, og du plasserer mer enn 40 prosent av anleggsmidlene i bruk i løpet av fjerde kvartal av året, må du bruke avskrivningskonvensjonen for midten av kvartalet.</span><span class="sxs-lookup"><span data-stu-id="91701-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="91701-106">Du kan bruke masseoppdateringsprosessen for å endre alle anleggsmidler som skal bruke den nye avskrivningskonvensjonen.</span><span class="sxs-lookup"><span data-stu-id="91701-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="91701-107">Når du oppdaterer avskrivningskonvensjonen for anleggsmidler, sletter du alle avskrivningstransaksjoner som finnes for disse anleggsmidlene.</span><span class="sxs-lookup"><span data-stu-id="91701-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="91701-108">Du sletter også alle transaksjoner for avskrivningsjusteringer, transaksjoner for bonusavskrivning og transaksjoner for ekstraordinær avskrivning for disse anleggsmidlene.</span><span class="sxs-lookup"><span data-stu-id="91701-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="91701-109">Når du skal oppdatere avskrivningskonvensjonen for anleggsmidler som allerede er avhendet, må du først slette de eksisterende avhendingstransaksjonene.</span><span class="sxs-lookup"><span data-stu-id="91701-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="91701-110">Du må også slette alle transaksjoner som ble generert på grunnlag av avhendingsprosessen.</span><span class="sxs-lookup"><span data-stu-id="91701-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="91701-111">Når du har oppdatert avskrivningskonvensjonen for anleggsmidler, kan du behandle avskrivning og ekstraordinær avskrivning for hvert anleggsmiddel.</span><span class="sxs-lookup"><span data-stu-id="91701-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="91701-112">Du kan også gjøre manuelle avskrivningsjusteringer hvis justeringer er nødvendig.</span><span class="sxs-lookup"><span data-stu-id="91701-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>

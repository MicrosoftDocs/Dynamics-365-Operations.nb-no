# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="a412b-103">Avhende et anleggsmiddel ved hjelp av en fritekstfaktura</span><span class="sxs-lookup"><span data-stu-id="a412b-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a412b-104">Dette emnet beskriver hvordan du kvitter deg med et anleggsmiddel ved hjelp av fritekstfakturaen.</span><span class="sxs-lookup"><span data-stu-id="a412b-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="a412b-105">I navigasjonsruten går du til **Moduler > Kunder > Fakturaer > Alle fritekstfakturaer**.</span><span class="sxs-lookup"><span data-stu-id="a412b-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="a412b-106">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="a412b-106">Select **New**.</span></span>
3. <span data-ttu-id="a412b-107">Angi eller velg en verdi i **Kundekonto**-feltet.</span><span class="sxs-lookup"><span data-stu-id="a412b-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="a412b-108">Valider standard **Faktura**-dato, og rediger om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="a412b-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="a412b-109">Valider gjenværende standard hodefelt, for eksempel **Valuta**, og rediger om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="a412b-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="a412b-110">Valider feltene **Beskrivelse** og **Hovedkonto** i delen **Fakturalinjer**.</span><span class="sxs-lookup"><span data-stu-id="a412b-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="a412b-111">Valider standardfeltene **mva-gruppe** og **mva-gruppe for varer**.</span><span class="sxs-lookup"><span data-stu-id="a412b-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="a412b-112">Angi **enhetsprisen** eller **salgsbeløpet** til anleggsmidlet.</span><span class="sxs-lookup"><span data-stu-id="a412b-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="a412b-113">Velg delen **Linjedetaljer**.</span><span class="sxs-lookup"><span data-stu-id="a412b-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="a412b-114">Angi eller velg en verdi i feltet **Anleggsmiddel**.</span><span class="sxs-lookup"><span data-stu-id="a412b-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="a412b-115">Velg **Poster**.</span><span class="sxs-lookup"><span data-stu-id="a412b-115">Select **Post**.</span></span>

# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="2bf51-103">Definere anleggsmiddelgrupper</span><span class="sxs-lookup"><span data-stu-id="2bf51-103">Set up fixed asset groups</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2bf51-104">Dette emnet forklarer hvordan du oppretter en ny anleggsmiddelgruppe.</span><span class="sxs-lookup"><span data-stu-id="2bf51-104">This topic explains how to create a new fixed asset group.</span></span> <span data-ttu-id="2bf51-105">Den bruker regnskapsførerrollen og demodataene for den juridiske enheten USMF.</span><span class="sxs-lookup"><span data-stu-id="2bf51-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="2bf51-106">I navigasjonsruten går du til **Moduler > Anleggsmidler > Oppsett > Anleggsmiddelgrupper**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-106">In the navigation pane, go to **Modules > Fixed assets > Setup > Fixed asset groups**.</span></span>
2. <span data-ttu-id="2bf51-107">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-107">Select **New**.</span></span>
3. <span data-ttu-id="2bf51-108">Skriv inn en verdi i feltet **Anleggsmiddelgruppe**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-108">In the **Fixed asset group** field, type a value.</span></span>
4. <span data-ttu-id="2bf51-109">Skriv inn en verdi i **Navn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="2bf51-109">In the **Name** field, type a value.</span></span> <span data-ttu-id="2bf51-110">Automatisk nummerering av anleggsmidler og nummerseriekode i **Anleggsmiddel**-gruppen overstyrer innstillingene i parametere for anleggsmidler.</span><span class="sxs-lookup"><span data-stu-id="2bf51-110">Autonumber fixed assets and Number sequence code on the **Fixed asset** group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="2bf51-111">Du kan endre det her hvis anleggsmidlene i denne anleggsmiddelgruppen skal ha forskjellig nummerering fra andre grupper.</span><span class="sxs-lookup"><span data-stu-id="2bf51-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="2bf51-112">Velg **Bøker**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-112">Select **Books**.</span></span>
6. <span data-ttu-id="2bf51-113">Angi eller velg en verdi i feltet **Tablå**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-113">In the **Book** field, enter or select a value.</span></span> <span data-ttu-id="2bf51-114">Feltet **Beregn avskrivning** er satt til **Ja**, så anleggsmiddeltablået inkluderes i avskrivningsforslag.</span><span class="sxs-lookup"><span data-stu-id="2bf51-114">The **Calculate depreciation** field is set to **Yes**, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="2bf51-115">Hvis **Beregn avskrivning** er satt til **Nei**, vil anleggsmidlet ikke automatisk avskrives.</span><span class="sxs-lookup"><span data-stu-id="2bf51-115">If **Calculate depreciation** is set to **No**, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="2bf51-116">Angi levetiden for anleggsmidlet i år.</span><span class="sxs-lookup"><span data-stu-id="2bf51-116">Set the Service life of the asset, in years.</span></span> <span data-ttu-id="2bf51-117">Merk at verdien i feltet **Avskrivningsperioder** beregnes etter at levetiden er angitt.</span><span class="sxs-lookup"><span data-stu-id="2bf51-117">Note that the **Depreciation periods** field value is calculated after setting the Service life.</span></span>  
8. <span data-ttu-id="2bf51-118">Velg et alternativ i feltet **Avskrivningskonvensjon**.</span><span class="sxs-lookup"><span data-stu-id="2bf51-118">In the **Depreciation convention** field, select an option.</span></span>
9. <span data-ttu-id="2bf51-119">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="2bf51-119">Close the page.</span></span>

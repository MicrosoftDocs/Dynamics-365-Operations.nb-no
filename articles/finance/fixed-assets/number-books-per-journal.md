# <a name="number-of-books-per-journal"></a><span data-ttu-id="62444-104">Antall tablåer per journal</span><span class="sxs-lookup"><span data-stu-id="62444-104">Number of books per journal</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="62444-105">Dette emnet beskriver relasjonen mellom journalene og aktivatablåene når du oppretter en anleggsmiddelanskaffelse eller et avskrivningsforslag via en satsvis jobb.</span><span class="sxs-lookup"><span data-stu-id="62444-105">This topic describes the relationship between journals and asset books when you create a fixed asset acquisition or depreciation proposal through a batch job.</span></span> <span data-ttu-id="62444-106">Du kan definere det maksimale antallet tablåer som er inkludert for hver anskaffelse og avskrivning ved hjelp av feltene i delen **Antall tablåer per journal** i kategorien **Generelt** på siden **Parametere for anleggsmidler** (**Anleggsmidler \> Oppsett \> Parametere for anleggsmidler**).</span><span class="sxs-lookup"><span data-stu-id="62444-106">You can define the maximum number of books that are included for each acquisition and for depreciation by using the fields in the **Number of books per journal** section on the **General** tab of the **Fixed assets parameters** page (**Fixed assets \> Setup \> Fixed assets parameters**).</span></span> <span data-ttu-id="62444-107">Med disse feltene kan du fordele antall anleggsmiddeltablåer per anskaffelsesjournal og avskrivningsjournal.</span><span class="sxs-lookup"><span data-stu-id="62444-107">These fields let you distribute the number of asset books per acquisition journal and depreciation journal.</span></span>

<span data-ttu-id="62444-108">For et anskaffelsesforslag er standardverdien minst 10 000 tablåer.</span><span class="sxs-lookup"><span data-stu-id="62444-108">For an acquisition proposal, the default value is at least 10,000 books.</span></span> <span data-ttu-id="62444-109">For et avskrivningsforslag er standardverdien minst 2000 tablåer.</span><span class="sxs-lookup"><span data-stu-id="62444-109">For a depreciation proposal, the default value is at least 2,000 books.</span></span>

<span data-ttu-id="62444-110">Hvis det for eksempel er 4 000 anleggsmidler, og to tablåer er knyttet til hvert anleggsmiddel, posteres ett tablå til det gjeldende laget, og det andre tablået posteres til skattelaget.</span><span class="sxs-lookup"><span data-stu-id="62444-110">For example, if there are 4,000 fixed assets, and two books are associated with each asset, one book will be posted to the current layer, and the other book will be posted to the tax layer.</span></span> <span data-ttu-id="62444-111">Hvis du anskaffer 4 000 anleggsmidler ved hjelp av satsvis behandling, vil den satsvise jobben som oppretter en anskaffelsesjournal for anleggsmidler, inneholde 4 000 aktivatablåer.</span><span class="sxs-lookup"><span data-stu-id="62444-111">If you acquire 4,000 fixed assets through batch processing, the batch job that creates one fixed asset acquisition journal will contain 4,000 asset books.</span></span>

> [!NOTE]
> <span data-ttu-id="62444-112">Journalen som opprettes, vil fortsatt bli brukt til den satsvise jobben er fullført.</span><span class="sxs-lookup"><span data-stu-id="62444-112">The journal that is created will continue to be used until the batch job is completed.</span></span>
>
> <span data-ttu-id="62444-113">De avledede tablåene er ikke inkludert i det maksimale antall tabler per journal.</span><span class="sxs-lookup"><span data-stu-id="62444-113">The derived books aren't included in the maximum number of books per journal.</span></span>

<span data-ttu-id="62444-114">Du kan bruke satsvis behandling til å kjøre avskrivning for samme settet med anskaffede aktivaer.</span><span class="sxs-lookup"><span data-stu-id="62444-114">You can use  batch processing to run depreciation for the same set of acquired assets.</span></span> <span data-ttu-id="62444-115">En satsvis jobb oppretter for eksempel to avskrivningsjournaler, som hver består av 2 000 aktivatablåer.</span><span class="sxs-lookup"><span data-stu-id="62444-115">For example, a batch job creates two depreciation journals, each of which consists of 2,000 asset books.</span></span> <span data-ttu-id="62444-116">Den første journalen vil inneholde tablåer som er knyttet til anleggsmidlene som er nummerert fra 1 til 2 000.</span><span class="sxs-lookup"><span data-stu-id="62444-116">The first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,000.</span></span> <span data-ttu-id="62444-117">Den andre journalen vil deretter inneholde tablåer som er knyttet til anleggsmidlene som er nummerert fra 2001 til og med 4000.</span><span class="sxs-lookup"><span data-stu-id="62444-117">The second journal will then contain books that are associated with the fixed assets that are numbered 2,001 through 4,000.</span></span>

<span data-ttu-id="62444-118">Jobben for satsvis behandling utelukker lukkede tablåer.</span><span class="sxs-lookup"><span data-stu-id="62444-118">The batch processing job excludes closed books.</span></span> <span data-ttu-id="62444-119">I en satsvis jobb for avskrivning, er for eksempel 10 i de første 2 000 tablåene lukket.</span><span class="sxs-lookup"><span data-stu-id="62444-119">For example, in a batch job for depreciation, 10 of the first 2,000 books are closed.</span></span> <span data-ttu-id="62444-120">I dette tilfellet vil den første journalen inneholde tablåer som er knyttet til anleggsmidlene som er nummerert fra 1 til og med 2011.</span><span class="sxs-lookup"><span data-stu-id="62444-120">In this case, the first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,011.</span></span> <span data-ttu-id="62444-121">Den andre journalen vil deretter inneholde tablåer som er knyttet til anleggsmidlene som er nummerert fra 2012 til og med 4000.</span><span class="sxs-lookup"><span data-stu-id="62444-121">The second journal will then contain books that are associated with the fixed assets that are numbered 2,012 through 4,000.</span></span>

> [!NOTE]
> <span data-ttu-id="62444-122">Hvis du har ID-er for anleggsmidler med forskjellige skilletegn (for eksempel – eller /), og du oppretter anleggsmiddeltransaksjoner i satsvise jobber, må du kjøre en egen satsvis jobb for hver type skilletegn.</span><span class="sxs-lookup"><span data-stu-id="62444-122">If you have fixed asset IDs with different separators (such as – or /), and you create fixed asset transactions in batch jobs, you must run a separate batch job for each type of separator.</span></span> <span data-ttu-id="62444-123">Systemet kan ikke behandle forskjellige skilletegn innenfor den samme satsvise jobben.</span><span class="sxs-lookup"><span data-stu-id="62444-123">The system can't process different separators within the same batch job.</span></span>

<span data-ttu-id="62444-124">Grensen for antall tabler brukes hvis dupliserte aktiva-IDer ikke finnes i samme journal.</span><span class="sxs-lookup"><span data-stu-id="62444-124">The limit on the number of books is applied if duplicate asset IDs don't exist in the same journal.</span></span> <span data-ttu-id="62444-125">Hvis aktiva-IDen er den samme som tablå-IDen, kan imidlertid antall tablåer per journal overstiges for å beholde aktiva-IDen i samme journal.</span><span class="sxs-lookup"><span data-stu-id="62444-125">However, if the asset ID is the same as the book ID, the number of books per journal can be exceeded to keep the asset ID in the same journal.</span></span>

<span data-ttu-id="62444-126">Det finnes for eksempel 5 001 anleggsmiddel-IDer, tre tablåer er knyttet til hver anleggsmiddel-ID, og hvert anleggsmiddeltablå posteres til det samme posteringslaget.</span><span class="sxs-lookup"><span data-stu-id="62444-126">For example, there are 5,001 fixed asset IDs, three books are associated with each fixed asset ID, and each asset book is posted to the same posting layer.</span></span> <span data-ttu-id="62444-127">Du kjører avskrivning for tre påfølgende måneder, uten sammendrag.</span><span class="sxs-lookup"><span data-stu-id="62444-127">You run depreciation for three consecutive months, without summarizing.</span></span>  <span data-ttu-id="62444-128">Avskrivningsjournalen blir opprettet ved hjelp av en satsvis jobb, og systemet vil opprette sju journaler som har 667 anleggsmiddel-IDer og tre tablåer for hver anleggsmiddel-ID.</span><span class="sxs-lookup"><span data-stu-id="62444-128">The depreciation journal will be created through a batch job, and the system will create seven journals that have 667 fixed asset IDs and three books for each fixed asset ID.</span></span> <span løpet av tre måneder vil det derfor være 6 003 journallinjer for å vedlikeholde de samme anleggsmiddel-IDene i samme journal.</span><span class="sxs-lookup"><span data-stu-id="62444-130">Therefore, in three months, there will be 6,003 journal lines to maintain the same asset IDs in the same journal.</span></span> <span data-ttu-id="62444-131">Systemet vil også opprette én journal som har 332 anleggsmiddel-IDer og tre tablåer for hver anleggsmiddel-ID.</span><span class="sxs-lookup"><span data-stu-id="62444-131">The system will also create one journal that has 332 fixed asset IDs and three books for each fixed asset ID.</span></span> <span data-ttu-id="62444-132">I løpet av tre måneder vil det være 2 988 linjer.</span><span class="sxs-lookup"><span data-stu-id="62444-132">In three months, there will be 2,988 lines.</span></span>

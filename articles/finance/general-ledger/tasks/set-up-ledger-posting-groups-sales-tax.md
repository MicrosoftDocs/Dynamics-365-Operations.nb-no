# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="8c10c-103">Konfigurere finansposteringsgrupper for merverdiavgift</span><span class="sxs-lookup"><span data-stu-id="8c10c-103">Set up Ledger posting groups for sales tax</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8c10c-104">Merverdiavgiften beregnes og posteres til hovedkontoer som er angitt i finansposteringsgrupper.</span><span class="sxs-lookup"><span data-stu-id="8c10c-104">Sales tax is calculated and posted to main accounts that are specified in Ledger posting groups.</span></span> <span data-ttu-id="8c10c-105">Finansposteringsgrupper er tilknyttet hver mva-kode.</span><span class="sxs-lookup"><span data-stu-id="8c10c-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="8c10c-106">Du kan angi individuelle finansposteringsgrupper for hver mva-kode, bruke én finansposteringsgruppe for alle mva-koder eller tilordne flere finansposteringsgrupper til mva-kodene.</span><span class="sxs-lookup"><span data-stu-id="8c10c-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="8c10c-107">Denne registreringen bruker demonstrasjonsfirmaet DEMF.</span><span class="sxs-lookup"><span data-stu-id="8c10c-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="8c10c-108">Gå til **Navigasjonsrute > Moduler > Avgift > Oppsett > Merverdiavgift > Finansposteringsgrupper**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="8c10c-109">Klikk på **Ny**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-109">Click **New**.</span></span>
3. <span data-ttu-id="8c10c-110">Skriv inn en verdi i feltet **Finansposteringsgruppe**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="8c10c-111">Skriv inn en verdi i **Beskrivelse**-feltet.</span><span class="sxs-lookup"><span data-stu-id="8c10c-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="8c10c-112">Velg hovedkontoen for utgående merverdiavgift som betales til skattemyndighetene, i feltet **Konto, utgående mva**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="8c10c-113">Merverdiavgift innkreves på vegne av skattemyndighetene når du selger avgiftspliktige varer og tjenester.</span><span class="sxs-lookup"><span data-stu-id="8c10c-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="8c10c-114">Velg hovedkontoen for inngående merverdiavgift som mottas fra skattemyndighetene, i feltet **Innkommende merverdiavgift**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="8c10c-115">Leverandører samler inn avgifter på vegne av skattemyndighetene når du kjøper avgiftspliktige varer og tjenester.</span><span class="sxs-lookup"><span data-stu-id="8c10c-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="8c10c-116">Dette feltet er ikke tilgjengelig hvis alternativet Bruk avgiftsregler for merverdiavgift er valgt på siden **Økonomiparametere**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="8c10c-117">I stedet blir mva som betales til leverandører, debitert til den samme kontoen som kjøpet.</span><span class="sxs-lookup"><span data-stu-id="8c10c-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="8c10c-118">I feltet **Use tax-utgift** velger du hovedkontoen for postering av fradragsberettiget mva som ikke er gjort krav på eller rapportert til skattemyndighetene av leverandørene som en del av EU snudd avregning for GST/HST.</span><span class="sxs-lookup"><span data-stu-id="8c10c-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="8c10c-119">Alternativet **Use tax** må velges for **mva-koden** i **mva-gruppen** som skal brukes i transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8c10c-119">The **Use tax** option must be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="8c10c-120">Dette feltet er ikke tilgjengelig hvis alternativet **Bruk avgiftsregler for merverdiavgift** er valgt på siden **Økonomiparametere**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-120">This field won't be available if the **Apply sales tax taxation rules** option is selected on the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="8c10c-121">Velg hovedkontoen for postering av inngående use tax som betales til skattemyndighetene, i feltet **Forfalt use tax**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="8c10c-122">Alternativet **Use tax** må velges i **mva-koden** i **mva-gruppen** for å kontere **Use tax**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-122">The **Use tax** option must be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="8c10c-123">Hvis alternativet **Bruk avgiftsregler for merverdiavgift** er valgt i **Økonomiparametere**, posteres forskyvningen til utgiftskonto for transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8c10c-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="8c10c-124">I **Utligningskonto**-feltet velger du hovedkontoen der nettosaldoen for finanskontoene angav i feltene **Forfalt use tax** og **Innkommende merverdiavgift**, blir postert.</span><span class="sxs-lookup"><span data-stu-id="8c10c-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="8c10c-125">Saldoen opprettes når du utligner merverdiavgiften og posteringsjobben er kjørt.</span><span class="sxs-lookup"><span data-stu-id="8c10c-125">The balance will be created when the sales taxes settle and the posting job has been run.</span></span>  <span data-ttu-id="8c10c-126">Hvis skattemyndighetene for utligningsperioden er knyttet til en leverandørkonto, posteres saldoen i stedet til leverandørkontoen.</span><span class="sxs-lookup"><span data-stu-id="8c10c-126">If the taxing authority for the settlement period is associated with a vendor account, the balance will be posted to the vendor account instead.</span></span>
10. <span data-ttu-id="8c10c-127">Velg hovedkontoen for å postere kontorabatt for mva-koder knyttet til denne finansposteringsgruppen, i feltet **Leverandørkontantrabatt**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="8c10c-128">Dette er valgfritt, og hvis ingen kontoer angis, brukes hovedkontoen i **kontantrabattkoder**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="8c10c-129">Det kan være nyttig å bruke ulike kontoer per **finansposteringsgruppe** hvis du bruker alternativet Tilbakefør merverdiavgift på kontantrabatt på mva-grupper.</span><span class="sxs-lookup"><span data-stu-id="8c10c-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="8c10c-130">Velg hovedkontoen for å postere kontorabatt for **mva-koder** knyttet til denne **finansposteringsgruppen**, i feltet **Kundekontantrabatt**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="8c10c-131">Dette er valgfritt, og hvis ingen kontoer angis, brukes hovedkontoen i **kontantrabattkoder**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="8c10c-132">Det kan være nyttig å bruke ulike kontoer per **finansposteringsgruppe** hvis du bruker alternativet Tilbakefør merverdiavgift på kontantrabatt på **mva-grupper**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="8c10c-133">Klikk **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="8c10c-133">Click **Save**.</span></span>

# <a name="create-a-main-account"></a><span data-ttu-id="cb70e-103">Opprette en hovedkonto</span><span class="sxs-lookup"><span data-stu-id="cb70e-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="cb70e-104">Denne oppgaveveiledningen går gjennom å legge til en hovedkonto i en eksisterende kontoplan.</span><span class="sxs-lookup"><span data-stu-id="cb70e-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="cb70e-105">Denne registreringen bruker demonstrasjonsfirmaet USMF.</span><span class="sxs-lookup"><span data-stu-id="cb70e-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="cb70e-106">Gå til **Navigasjonsrute > Moduler > Økonomimodul > Kontoplan > Kontoer > Hovedkontoer**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="cb70e-107">Klikk på **Ny**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-107">Click **New**.</span></span>
3. <span data-ttu-id="cb70e-108">Skriv inn en verdi i feltet **Hovedkonto**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="cb70e-109">Skriv inn en verdi i **Navn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb70e-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="cb70e-110">Velg typen som best representerer kontosaldoen og lokasjonen i regnskapsoppgjør, i feltet **Hovedkontotype**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="cb70e-111">Velg kontokategorien som hovedkontoen tilhører, i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="cb70e-112">Kontokategorien brukes for standard finansrapporter og Power BI-instrumentbordinnhold.</span><span class="sxs-lookup"><span data-stu-id="cb70e-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="cb70e-113">Klikk koblingen i den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="cb70e-114">Endre standard debet eller kredit saldoen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="cb70e-115">Velg en verdi fra listen over valutaer i **Standardvaluta**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb70e-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="cb70e-116">Finn og velg ønsket post i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="cb70e-117">Klikk koblingen i den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="cb70e-118">Aktiver/deaktiver utvidelsen av delen **Overstyringer for juridisk enhet**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="cb70e-119">Klikk på **Legg til** for å velge en juridisk enhet.</span><span class="sxs-lookup"><span data-stu-id="cb70e-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="cb70e-120">Velg juridisk enhet i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="cb70e-121">Klikk **Legg til**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-121">Click **Add**.</span></span>
15. <span data-ttu-id="cb70e-122">Merk den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="cb70e-123">Merk av eller fjern merket i boksen **Avbrutt**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="cb70e-124">Vis delen **Finansrapportering**.</span><span class="sxs-lookup"><span data-stu-id="cb70e-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="cb70e-125">Klikk rullegardinknappen i feltet **Valutakurstype** for å åpne oppslaget.</span><span class="sxs-lookup"><span data-stu-id="cb70e-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="cb70e-126">Velg **valutakurstypen for kontoen** i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="cb70e-127">Klikk koblingen i den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="cb70e-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="cb70e-128">Velg metoden for å beregne valutakurser for kontoen i **Type valutaveksling**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb70e-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="cb70e-129">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="cb70e-129">Close the page.</span></span>

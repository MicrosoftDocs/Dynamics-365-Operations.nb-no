# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a><span data-ttu-id="8dc70-103">Vise posterte TDS-betalinger og -transaksjoner for en TDS-utligningsperiode</span><span class="sxs-lookup"><span data-stu-id="8dc70-103">View posted TDS payments and transactions for a TDS settlement period</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8dc70-104">Dette emnet forklarer hvordan du kan vise TDS-betalingene og -transaksjonene (Tax Deducted at Source) som ble postert for en utligningsperiode.</span><span class="sxs-lookup"><span data-stu-id="8dc70-104">This topic explains how to view the Tax Deducted at Source (TDS) payments and transactions that were posted for a settlement period.</span></span>

1. <span data-ttu-id="8dc70-105">Gå til **Avgift \> Indirekte avgifter \> Kildeskatt \> Utligningsperioder for kildeskatt**.</span><span class="sxs-lookup"><span data-stu-id="8dc70-105">Go to **Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**.</span></span>

    <span data-ttu-id="8dc70-106">[![Siden Utligningsperioder for kildeskatt](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span><span class="sxs-lookup"><span data-stu-id="8dc70-106">[![Withholding tax settlement periods page](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span></span>

2. <span data-ttu-id="8dc70-107">Velg **Kildeskattbetalinger** på siden **Utligningsperioder for kildeskatt** for å åpne siden **Kildeskattbetalinger**, der du kan vise TDS-utligningene som ble foretatt for en bestemt TDS-utligningsperiode.</span><span class="sxs-lookup"><span data-stu-id="8dc70-107">On the **Withholding tax settlement periods** page, select **Withholding tax payments** to open the **Withholding tax payments** page, where you can view the TDS settlements that were made for a specific TDS settlement period.</span></span>

    <span data-ttu-id="8dc70-108">**Oversikt**-fanen viser følgende informasjon:</span><span class="sxs-lookup"><span data-stu-id="8dc70-108">The **Overview** tab shows the following information:</span></span>

    - <span data-ttu-id="8dc70-109">**Dato** – Datoen for TDS-utligning.</span><span class="sxs-lookup"><span data-stu-id="8dc70-109">**Date** – The date of TDS settlement.</span></span>
    - <span data-ttu-id="8dc70-110">**Bilag** – Bilagsnummeret for TDS-utligningstransaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8dc70-110">**Voucher** – The voucher number of the TDS settlement transaction.</span></span>
    - <span data-ttu-id="8dc70-111">**Avgiftstype** – Avgiftstypen som utligningsprosessen kjøres for.</span><span class="sxs-lookup"><span data-stu-id="8dc70-111">**Tax type** – The tax type that the settlement process is run for.</span></span>
    - <span data-ttu-id="8dc70-112">**TAN-nummer (Tax Account Number)** – TAN-nummeret (Tax Account Number) for den utlignede TDS-transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8dc70-112">**Tax Account Number (TAN)** – The Tax Account Number (TAN) of the settled TDS transaction.</span></span>
    - <span data-ttu-id="8dc70-113">**Utligningsperiode** – Utligningsperioden som TDS-utligningsprosessen kjøres for.</span><span class="sxs-lookup"><span data-stu-id="8dc70-113">**Settlement period** – The settlement period that the TDS settlement process is run for.</span></span>
    - <span data-ttu-id="8dc70-114">**Fra-dato** – Startdatoen for utligningsperioden.</span><span class="sxs-lookup"><span data-stu-id="8dc70-114">**From date** – The start date of the settlement period.</span></span>
    - <span data-ttu-id="8dc70-115">**Til-dato** – Sluttdatoen for utligningsperioden.</span><span class="sxs-lookup"><span data-stu-id="8dc70-115">**To date** – The end date of the settlement period.</span></span>
    - <span data-ttu-id="8dc70-116">**Kildeskattbetalingsversjon** – Versjonen av kildeskattbetalingen: **Opprinnelig**, **Siste rettelser** eller **Total liste**.</span><span class="sxs-lookup"><span data-stu-id="8dc70-116">**Withholding tax payment version** – The version of the withholding tax payment: **Original**, **Latest corrections**, or **Total list**.</span></span>

5. <span data-ttu-id="8dc70-117">Velg **Bilag** for å vise bilagsoppføringene for TDS-transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8dc70-117">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span>
6. <span data-ttu-id="8dc70-118">Velg **Kildeskattransaksjoner** for å vise detaljene for TDS-transaksjonene som ble utlignet for en bestemt periode i en utligningsperiode.</span><span class="sxs-lookup"><span data-stu-id="8dc70-118">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for a specific period during a settlement period.</span></span> <span data-ttu-id="8dc70-119">Velg **Bilag** for å vise bilagsoppføringene for TDS-transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="8dc70-119">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span> <span data-ttu-id="8dc70-120">Velg **Komponenter for kildeskatt** for å vise TDS-beløpet som ble beregnet per TDS-avgiftskomponent for en bestemt TDS-avgiftskode.</span><span class="sxs-lookup"><span data-stu-id="8dc70-120">Select **Withholding tax components** to view the TDS that was calculated per TDS tax component for a specific TDS tax code.</span></span>
7. <span data-ttu-id="8dc70-121">Lukk siden **Kildeskattbetalinger** for å gå tilbake til siden **Utligningsperioder for kildeskatt**.</span><span class="sxs-lookup"><span data-stu-id="8dc70-121">Close the **Withholding tax payments** page to return to the **Withholding tax settlement periods** page.</span></span>
8. <span data-ttu-id="8dc70-122">Velg **Kildeskattransaksjoner** for å vise detaljene for TDS-transaksjonene som ble utlignet for utligningsperioden.</span><span class="sxs-lookup"><span data-stu-id="8dc70-122">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for the settlement period.</span></span>

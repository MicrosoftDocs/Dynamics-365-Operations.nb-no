# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="9f80a-103">Vis en reiseregning fra økonomimodulen eller Innkjøp og leverandører</span><span class="sxs-lookup"><span data-stu-id="9f80a-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9f80a-104">Når du kjører en spørring for å vise transaksjoner i økonomimodulen eller Innkjøp og leverandører, kan spørringen gi for mye informasjon.</span><span class="sxs-lookup"><span data-stu-id="9f80a-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="9f80a-105">Det kan derfor hende at du vil begrense søket.</span><span class="sxs-lookup"><span data-stu-id="9f80a-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="9f80a-106">For å få informasjonen du trenger, kan du drille ned til en transaksjon for å se reiseregningen som den vises på.</span><span class="sxs-lookup"><span data-stu-id="9f80a-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="9f80a-107">For utgiftsrapporter kan du drille ned fra en transaksjon til den opprinnelige reiseregningen som er lagret i Reiseregning og utlegg.</span><span class="sxs-lookup"><span data-stu-id="9f80a-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="9f80a-108">I tillegg til det totale utgiftsbeløpet kan du vise de detaljerte transaksjonene, transaksjonene som er kvalifisert for merverdiavgiftsrefusjon (mva) og kvitteringene som er knyttet til utgiftene.</span><span class="sxs-lookup"><span data-stu-id="9f80a-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="9f80a-109">For å drille ned til den opprinnelige reiseregningen som en transaksjon vises på, velger du transaksjonen, og deretter **Originaldokumentet**.</span><span class="sxs-lookup"><span data-stu-id="9f80a-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="9f80a-110">Når du ser på individuelle transaksjoner, kan du også se tilleggsdata som er knyttet til en transaksjon.</span><span class="sxs-lookup"><span data-stu-id="9f80a-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="9f80a-111">Du kan for eksempel se en detaljert oversikt over hotellutgiftene, for eksempel Internett-tilgang, telefonsamtaler og room-service.</span><span class="sxs-lookup"><span data-stu-id="9f80a-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>

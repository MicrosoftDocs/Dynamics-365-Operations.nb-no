# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="0290b-103">Tilordne en kostnadselementdimensjon</span><span class="sxs-lookup"><span data-stu-id="0290b-103">Map a cost element dimension</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0290b-104">En kostnadskontroller kan bruke denne fremgangsmåten til å tilordne en kostnadselementdimensjon til en kostnadselementdimensjon i den juridiske MXMF-enheten.</span><span class="sxs-lookup"><span data-stu-id="0290b-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="0290b-105">Denne registreringen bruker USP2-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="0290b-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="0290b-106">Gå til Kostnadsregnskap > Dimensjoner > Kostnadselementdimensjoner.</span><span class="sxs-lookup"><span data-stu-id="0290b-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="0290b-107">Finn og velg ønsket post i listen.</span><span class="sxs-lookup"><span data-stu-id="0290b-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="0290b-108">Velg Kostnadselementer for dette eksemplet.</span><span class="sxs-lookup"><span data-stu-id="0290b-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="0290b-109">Klikk Dimensjonstilordninger.</span><span class="sxs-lookup"><span data-stu-id="0290b-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="0290b-110">Klikk Konfigurer tilordninger fra denne dimensjonen.</span><span class="sxs-lookup"><span data-stu-id="0290b-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="0290b-111">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="0290b-111">Click New.</span></span>
6. <span data-ttu-id="0290b-112">Angi eller velg en verdi i feltet Til dimensjon.</span><span class="sxs-lookup"><span data-stu-id="0290b-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="0290b-113">Velg MXMF-kostnadselementer for dette eksemplet.</span><span class="sxs-lookup"><span data-stu-id="0290b-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="0290b-114">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="0290b-114">Click New.</span></span>
8. <span data-ttu-id="0290b-115">Merk den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="0290b-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="0290b-116">Angi eller velg en verdi i feltet Fra dimensjonsmedlem.</span><span class="sxs-lookup"><span data-stu-id="0290b-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="0290b-117">For dette eksemplet, velg dimensjonsmedlemmet 606400 Telefon og faksutgift.</span><span class="sxs-lookup"><span data-stu-id="0290b-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="0290b-118">Angi eller velg en verdi i feltet Til dimensjonsmedlem.</span><span class="sxs-lookup"><span data-stu-id="0290b-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="0290b-119">For dette eksemplet, velg dimensjonsmedlemmet 6001004 Telefono.</span><span class="sxs-lookup"><span data-stu-id="0290b-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="0290b-120">Klikk Lagre.</span><span class="sxs-lookup"><span data-stu-id="0290b-120">Click Save.</span></span>

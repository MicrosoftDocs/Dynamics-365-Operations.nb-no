<span data-ttu-id="2c17d-105">**Tildelingsgrunnlag**</span><span class="sxs-lookup"><span data-stu-id="2c17d-105">**Allocation base**</span></span>

<span data-ttu-id="2c17d-106">Tildelingsgrunnlag brukes til å måle og kvantifisere aktiviteter, for eksempel maskintimer som brukes, kilowatt-timer som forbrukes eller kvadratfot som er opptatt.</span><span class="sxs-lookup"><span data-stu-id="2c17d-106">The allocation base is used to measure and quantify activities, such as machine hours that are used, kilowatt hours that are consumed, or square footage that is occupied.</span></span> <span data-ttu-id="2c17d-107">Den brukes som grunnlag for allokering av kostnader til en eller flere kostnadsobjekter</span><span class="sxs-lookup"><span data-stu-id="2c17d-107">It's used as basis for allocating costs to one or more cost objects</span></span>

<span data-ttu-id="2c17d-108">**Kostnadsregnskap**</span><span class="sxs-lookup"><span data-stu-id="2c17d-108">**Cost accounting**</span></span>

<span data-ttu-id="2c17d-109">Med Kostnadsregnskap kan du samle inn data fra forskjellige kilder, for eksempel i økonomimodulen, underfinans, budsjetter og statistisk informasjon.</span><span hverandre i Kostnadsregnskap.</span><span class="sxs-lookup"><span data-stu-id="2c17d-111">The source data that is used for cost analysis is treated independently in Cost accounting.</span></span> <span data-ttu-id="2c17d-112">Oppdateringer i Kostnadsregnskap påvirker derfor ikke kildedataene.</span><span class="sxs-lookup"><span data-stu-id="2c17d-112">Therefore, updates in Cost accounting <span data-ttu-id="2c17d-115">**Kostnadsregnskapsfinans**</span><span class="sxs-lookup"><span data-stu-id="2c17d-115">**Cost accounting ledger**</span></span>

<span data-ttu-id="2c17d-120">Enhver type objekt som er valgt for kostnadskontroll.</span><span class="sxs-lookup"><span data-stu-id="2c17d-120">Any type of object that is selected for cost control.</span></span> <span data-ttu-id="2c17d-121">Kostnader eller inntekter er enten direkte postet på eller allokert til kostnadsobjekter.</span><span class="sxs-lookup"><span data-stu-id="2c17d-121">Costs or revenues are either directly posted on or allocated to cost objects.</span></span> <span data-ttu-id="2c17d-122">Noen typiske kostnadsobjekter er:</span><span class="sxs-lookup"><span data-stu-id="2c17d-122">Some typical cost objects are:</span></span>

-  <span data-ttu-id="2c17d-124">Prosjekter</span><span class="sxs-lookup"><span data-stu-id="2c17d-124">Projects</span></span>
-  <span data-ttu-id="2c17d-125">Avdelinger</span><span class="sxs-lookup"><span data-stu-id="2c17d-125">Departments</span></span>
-  <span data-ttu-id="2c17d-126">Kostsentre</span><span class="sxs-lookup"><span data-stu-id="2c17d-126">Cost centers</span></span>

<span data-ttu-id="2c17d-127">Ledelsen bruker kostnadsobjekter til å kvantifisere kostnader, men også til å utføre fortjenesteanalyse.</span><span class="sxs-lookup"><span data-stu-id="2c17d-127">Management uses cost objects to quantify costs, but also to drive profitability analysis.</span></span>

<span data-ttu-id="2c17d-128">**Kostnadselement**</span><span class="sxs-lookup"><span data-stu-id="2c17d-128">**Cost element**</span></span>

<span data-ttu-id="2c17d-129">Brukes som en funksjon for å spore og kategorisere kostnader.</span><span class="sxs-lookup"><span data-stu-id="2c17d-129">Used as a function to track and categorize costs.</span></span> <span data-ttu-id="2c17d-130">Det er to typer kostnadselementer: primær og sekundær.</span><span class="sxs-lookup"><span data-stu-id="2c17d-130">There are two types of cost elements: primary and secondary.</span></span>

-   <span data-ttu-id="2c17d-139">**Etter elementer** – materialer, arbeid og utgifter.</span><span class="sxs-lookup"><span data-stu-id="2c17d-139">**By elements** – Materials, labor, and expenses.</span></span>

<span data-ttu-id="2c17d-145">**Kostnadsatferd**</span><span class="sxs-lookup"><span data-stu-id="2c17d-145">**Cost behavior**</span></span>

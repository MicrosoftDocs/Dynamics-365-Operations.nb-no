# <a name="customize-image-focal-points"></a><span data-ttu-id="c1dff-103">Tilpasse bildefokuspunkter</span><span class="sxs-lookup"><span data-stu-id="c1dff-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="c1dff-104">Dette emnet beskriver hvordan du tilpasser bildefokuspunkter i områdebygger for Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="c1dff-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="c1dff-105">Oversikt</span><span class="sxs-lookup"><span data-stu-id="c1dff-105">Overview</span></span>

<span data-ttu-id="c1dff-106">Når et bilde lastes opp til mediebiblioteket for Commerce-områdebygger, prøver systemet å bestemme bildets fokuspunkt.</span><span class="sxs-lookup"><span data-stu-id="c1dff-106">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="c1dff-107">Hvis for eksempel bildet har en person på seg, vil systemet sette fokuspunktet til personens ansikt som er standard.</span><span class="sxs-lookup"><span data-stu-id="c1dff-107">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="c1dff-108">I de fleste tilfeller fungerer fokuspunktet automatisk bra for alle visningsporter, men noen ganger vil du kanskje justere fokuspunktet for å sikre at en bestemt del av bildet alltid er synlig.</span><span class="sxs-lookup"><span data-stu-id="c1dff-108">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="c1dff-109">Definere et egendefinert fokuspunkt for et bilde</span><span class="sxs-lookup"><span data-stu-id="c1dff-109">Define a custom focal point for an image</span></span>

<span data-ttu-id="c1dff-110">Følg denne fremgangsmåten for å definere et egendefinert fokuspunkt for et bilde:</span><span class="sxs-lookup"><span data-stu-id="c1dff-110">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="c1dff-111">Velg **Mediebibliotek** i den venstre navigasjonsruten i Commerce-områdebygger.</span><span class="sxs-lookup"><span data-stu-id="c1dff-111">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="c1dff-112">I hovedvinduet velger du bildet du vil endre.</span><span class="sxs-lookup"><span data-stu-id="c1dff-112">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="c1dff-113">Velg **Rediger** på kommandolinjen for å sjekke ut filen.</span><span class="sxs-lookup"><span data-stu-id="c1dff-113">On the command bar, select **Edit** to check out the file.</span></span>
1. <span data-ttu-id="c1dff-114">Velg bildet for å gå til **Redigeringsmodus**.</span><span class="sxs-lookup"><span data-stu-id="c1dff-114">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="c1dff-115">Velg **Endre fokuspunkt** under **Redigeringsmoudus**.</span><span class="sxs-lookup"><span data-stu-id="c1dff-115">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="c1dff-116">En sirkelformet fokuspunktkontroll vises over bildet.</span><span class="sxs-lookup"><span data-stu-id="c1dff-116">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="c1dff-117">Velg fokuspunktkontrollen for å flytte den over det ønskede fokuspunktet.</span><span class="sxs-lookup"><span data-stu-id="c1dff-117">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="c1dff-118">Når du er ferdig, velger du **Lagre** og deretter **Fullfør redigering** på kommandolinjen.</span><span class="sxs-lookup"><span data-stu-id="c1dff-118">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>

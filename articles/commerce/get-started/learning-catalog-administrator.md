# <a name="learning-catalog-for-dynamics-365-commerce-administrators"></a>Opplæringskatalog for Dynamics 365 Commerce-administratorer

Trenger du å holde systemer og data i flyt, klargjort og sikre døgnet rundt?

Katalogen nedenfor er ordnet fra kjernekunnskap til bestemte områder, og fra de fleste grunnleggende til de mest avanserte. Vi informerer deg hvis det finnes innhold i flere formater, slik at du kan velge opplæringsformatet som oppfyller dine behov. Denne katalogen inneholder bare funksjoner som er spesifikke for Dynamics 365 Commerce. Hvis du vil ha opplæring om funksjoner som deles med andre økonomi- og driftsapper, kan du se [Opplæringskatalog for Dynamics 365 Finance](../../finance/get-started/learning-catalog-administrator.md).

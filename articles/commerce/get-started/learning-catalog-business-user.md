# <a name="learning-catalog-for-users-of-dynamics-365-commerce"></a>Opplæringskatalog for brukere av Dynamics 365 Commerce

Har du nettopp fått et nytt program å bruke?

Katalogen nedenfor er ordnet fra kjernekunnskap til bestemte områder, og fra de fleste grunnleggende til de mest avanserte. Vi informerer deg hvis det finnes innhold i flere formater, slik at du kan velge opplæringsformatet som oppfyller dine behov.

Hvis du vil ha opplæring om funksjoner som deles med andre økonomi- og driftsapper, kan du se [Opplæringskatalog for Dynamics 365 Finance](../../finance/get-started/learning-catalog-business-user.md).

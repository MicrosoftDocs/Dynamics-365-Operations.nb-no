# <a name="commerce-chat-features-overview"></a>Oversikt over Commerce-nettpratfunksjoner

[!include [banner](includes/banner.md)]

Ved å integrere Microsoft Dynamics 365 Omnikanal for Customer Service- og Power Virtual Agents-programmer med Dynamics 365 Commerce-e-handelswebområder, kan selskaper gi kundene muligheten til å handle via nettprat. Denne artikkelen inneholder en oversikt over Commerce-nettpratfunksjonene som er tilgjengelige for nettshoppere når Omnikanal for Customer Service og Power Virtual Agents er integrert med Dynamics 365 Commerce-e-handelsnettsteder.

## <a name="omnichannel-for-customer-service"></a>Omnikanal for Customer Service

Omnikanal for Customer Service er et robust program som utvider kraften i Dynamics 365 Customer Service. Den gjør det mulig for organisasjoner å knytte til og engasjere seg med kundene via kanaler, for eksempel direktechat, tale og SMS (Short Message Service). Hvis du vil ha mer informasjon om Omnikanal for Customer Service, kan du se [Innføring i Omnikanal for Customer Service](/dynamics365/customer-service/introduction-omnichannel). Hvis du vil ha informasjon om hvordan du integrerer Omnikanal for Customer Service med en Dynamics 365 Commerce-webområde for e-handel, kan du se [Commerce-nettprat med Omnikanal for Customer Service-modul](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents lar deg opprette omfattende AI-drevne samtaleboter for en rekke forespørsler. Hvis du vil ha mer informasjon om Power Virtual Agents, kan du se [Power Virtual Agents-oversikt](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Hvis du vil ha informasjon om hvordan du integrerer Power Virtual Agents med et Dynamics 365 Commerce-webområde for e-handel, kan du se [Commerce-nettprat med Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents er allerede inkludert i Omnikanal for Customer Service. Hvis du trenger både virtuell agent (samtalebot)- og live agent-funksjoner, bruker du Omnikanal for Customer Service.

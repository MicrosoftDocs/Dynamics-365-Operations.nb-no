# <a name="ways-to-add-content"></a>Måter å legge til innhold på

[!include [banner](includes/banner.md)]

Denne artikkelen gir en oversikt over og koblinger til dokumentasjon om hvordan du kan behandle innhold ved hjelp av webredigeringsverktøysettet for områdebygger for Microsoft Dynamics 365 Commerce.

Det er mange måter å endre utseendet, følelsen og innholdet på området på. Avhengig av det nødvendige tilpassingsnivået kan mange av disse endringene implementeres av ikke-utviklere i områdebygger, webredigeringsverktøysettet som følger med Dynamics 365 Commerce. Ved hjelp av områdebygger kan du bygge maler, velge temaer samt velge og konfigurere moduler uten å skrive kode. Utviklingsferdigheter kreves derimot for å opprette nye temaer eller moduler, fordi arbeidsflyten for SDK for e-handel og Microsoft Dynamics Lifecycle Services-arbeidsflytdistribusjonen (LCS) må brukes.

Følgende artikler er gode utgangspunkt for å begynne å forstå hvordan du legger til og administrerer områdeinnhold. De fleste artiklene som er oppført, fokuserer på områder som ikke krever en utvikler. Noen omhandler grunnleggende innholdsredigering, mens andre fokuserer på oppgaver for områdeadministrator. Hver av disse artiklene vil angi at bestemte oppgaver kan kreve SDK-arbeid. Hver artikkel forutsetter at du allerede har klargjort et område og fått tilgang til områdebyggerverktøysettet for området ditt.

Velg en av følgende artikler for å komme i gang.

- Hvis du vil gjøre deg kjent med terminologien for innholdsbehandling som brukes i områdebygger og i denne dokumentasjonen, kan du se [Ordliste for sidemodell](page-elements-overview.md).
- Hvis du vil vite hvordan moduler fungerer i arbeidsflyter for innholdsbehandling, kan du se [Arbeide med moduler](work-with-modules.md).
- Hvis du vil endre teksten, bildene eller videoen på en eksisterende områdeside, kan du se [Arbeide med moduler](work-with-modules.md).
- Hvis du vil se hvordan fragmenter kan gjøre innholdsbehandling mer effektiv og fleksibel, se [Arbeide med fragmenter](work-with-fragments.md).
- For å bidra til å sikre en vellykket merkevarebasert redigeringsopplevelse for webinnholdsforfattere, kan du se [Oversikt over maler og oppsett](templates-layouts-overview.md) og [Arbeide med maler](work-with-templates.md).
- Hvis du vil ordne delene på nytt på en områdeside, se [Arbeide med oppsett](work-with-layouts.md).
- Hvis du vil endre skrifter, farger og generell utforming av områdesider, kan du se [Velge et områdetema](select-site-theme.md) eller [Arbeide med CSS-overstyringsfiler](css-override-files.md).
- Hvis du vil omorganisere eller legge til nye navigasjonsalternativer, kan du se [Tilpasse områdenavigering](customize-site-navigation.md).
- Hvis du vil lære hvordan du kan sette opp, forhåndsvise og publisere mange samtidige webinnholdsendringer, kan du se [Arbeide med publiseringsgrupper](publish-groups.md).

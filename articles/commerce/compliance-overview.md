# <a name="compliance-overview"></a>Samsvarsoversikt


[!include [banner](includes/banner.md)]

Dette emnet er en oversikt over samsvarsveiledningen som leveres med Microsoft Dynamics 365 Commerce, for å hjelpe deg med å gjøre firmaets nettsteder kompatible. Etterlevelse er en viktig faktor for alle virksomheter, fordi det hjelper brukerbasen med å koble til deres områdeinnhold. Gjennom etterlevelse kan du også bidra til å beskytte din bedrift fra dyre rettslige skritt eller høye bøter.

Samsvarsdokumentasjonen inneholder en gjennomgang av områder du bør vurdere når du bruker Dynamics 365 Commerce, for å hjelpe deg med å oppfylle firmaets samsvarskrav.

Du er ansvarlig for å se gjennom firmaets samsvarskrav og for redigering og bygging av områder som oppfyller standardene for disse kravene. Emnene nedenfor gir eksempler som viser hvordan du kan ta hensyn til overholdelse når du bruker redigeringsverktøyene for handel.

Hvis du vil vite mer om de grunnleggende prinsippene som Microsoft bruker, kan du gå til [Microsoft Trust Center](https://www.microsoft.com/trust-center). På dette området kan du også få mer informasjon om samsvarsområder.

[!include [banner](includes/banner.md)]

## <a name="install-the-one-click-designer"></a><span data-ttu-id="973d6-109">Installere ettklikksutformingen</span><span class="sxs-lookup"><span data-stu-id="973d6-109">Install the one-click designer</span></span>

1. <span data-ttu-id="973d6-110">I Commerce bruker du menyen øverst til venstre til å navigere til **Retail og Commerce** &gt; **Kanaloppsett** &gt; **Salgsstedsoppsett** &gt; **Salgssted** &gt; **Skjermoppsett**.</span><span class="sxs-lookup"><span data-stu-id="973d6-110">In Commerce, use the menu in the upper left to navigate to **Retail and Commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>

4. <span data-ttu-id="973d6-114">I meldingsboksen **Programinstallasjon – sikkerhetsvarsel** som vises, klikker du **Kjør** for å installere utformingsverten for detaljhandel.</span><span class="sxs-lookup"><span data-stu-id="973d6-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="973d6-115">En fremdriftsindikator viser fremdriften for installasjonen.</span><span class="sxs-lookup"><span data-stu-id="973d6-115">A progress indicator shows the progress of the installation.</span></span>
5. <span data-ttu-id="973d6-116">Når installasjonen er fullført, angir du Commerce-brukernavnet og -passordet på **Logg på**-siden, og deretter klikker du på **Logg på** for å starte utformingen.</span><span class="sxs-lookup"><span data-stu-id="973d6-116">After the installation is completed, on the **Sign in** page, enter your Commerce user name and password, and then click **Sign in** to start the designer.</span></span>

## <a name="troubleshoot-the-installation-of-the-layout-designer"></a><span data-ttu-id="973d6-119">Feilsøking i forbindelse med installasjon av utformingen av oppsett</span><span class="sxs-lookup"><span data-stu-id="973d6-119">Troubleshoot the installation of the Layout designer</span></span>

- <span data-ttu-id="973d6-120">Når du klikker **Utforming**, vises ikke meldingen om å laste ned (eller kjøre) installasjonsprogrammet, eller gjeldende sikkerhetsinnstillinger tillater ikke at du laster ned filen.</span><span class="sxs-lookup"><span data-stu-id="973d6-120">When you click **Designer**, the prompt to download (or run) the installer doesn't appear, or your current security settings don't allow you to download the file.</span></span> 

    <span data-ttu-id="973d6-121">**Løsninger:**</span><span class="sxs-lookup"><span data-stu-id="973d6-121">**Solutions:**</span></span>

    - <span data-ttu-id="973d6-124">I Internet Explorer legger du til URL-adressen for Commerce i klarerte områder.</span><span class="sxs-lookup"><span data-stu-id="973d6-124">In Internet Explorer, add the Commerce URL to your trusted sites.</span></span> <span data-ttu-id="973d6-125">Klikk **Innstillinger** &gt; **Alternativer** &gt; **Sikkerhet** &gt; **Klarerte områder** &gt; **Områder** &gt; **Legg til**.</span><span class="sxs-lookup"><span data-stu-id="973d6-125">Click **Settings** &gt; **Options** &gt; **Security** &gt; **Trusted sites** &gt; **Sites** &gt; **Add**.</span></span>

- <span data-ttu-id="973d6-126">Programmet starter ikke, og du blir bedt om å kontakte leverandøren.</span><span class="sxs-lookup"><span data-stu-id="973d6-126">The program doesn't start, and you're instructed to contact the vendor.</span></span>

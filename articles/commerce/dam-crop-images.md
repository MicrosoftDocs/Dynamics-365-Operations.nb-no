## <a name="overview"></a><span data-ttu-id="4f1ef-105">Oversikt</span><span class="sxs-lookup"><span data-stu-id="4f1ef-105">Overview</span></span>

<span data-ttu-id="4f1ef-106">Med mediebiblioteket for Commerce-områdebygger kan du beskjære bilder for å optimalisere dem for ulike modultyper og visningsporter.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="4f1ef-107">Beskjære et bilde</span><span class="sxs-lookup"><span data-stu-id="4f1ef-107">Crop an image</span></span>

<span data-ttu-id="4f1ef-108">Hvis du vil beskjære et bilde i områdebygger, følger du disse trinnene.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="4f1ef-109">Velg **Mediebibliotek** i den venstre navigasjonsruten i Commerce-områdebygger.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="4f1ef-110">I hovedvinduet velger du bildet du vil endre.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="4f1ef-111">På kommandolinjen velger du **Rediger**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="4f1ef-112">Velg bildet for å gå til **Redigeringsmodus**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="4f1ef-113">Under **Redigeringsmodus** velger du **Rediger visning etter modul**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="4f1ef-114">Velg modultypen fra rullegardinmenyen **Modul**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="4f1ef-115">Velg visningstypen fra rullegardinmenyen **Visningstype**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="4f1ef-116">Velg bildeplasseringen fra rullegardinmenyen **Plassering**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="4f1ef-117">Velg visningsstørrelsen fra rullegardinmenyen **Visningsport**.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="4f1ef-118">Bildet overlappes av området som representerer beskjæringsområdet.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="4f1ef-119">Flytt og endre størrelsen på beskjæringsområdet etter behov.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="4f1ef-120">Størrelsesforholdet blir vedlikeholdt automatisk.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="4f1ef-121">Når du er ferdig, velger du **Lagre** og deretter **Fullfør redigering** på kommandolinjen.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="4f1ef-122">Når egendefinert beskjæring er fullført, vil bildeendringer tre i kraft nesten umiddelbart.</span><span class="sxs-lookup"><span data-stu-id="4f1ef-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>

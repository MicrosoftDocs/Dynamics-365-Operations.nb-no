[!include [banner](includes/banner.md)]

<span data-ttu-id="92088-104">Dette emnet gir en oversikt over oppgavebehandling for ledere og arbeidere i Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="92088-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="92088-105">Oversikt</span><span class="sxs-lookup"><span data-stu-id="92088-105">Overview</span></span>

<span data-ttu-id="92088-106">I et detaljhandelsmiljø er det alltid vanskelig å sørge for at oppgavene utføres av den riktige personen til riktig tid.</span><span class="sxs-lookup"><span data-stu-id="92088-106">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="92088-107">Forhandlerne må kunne varsle arbeidere om kommende oppgaver og gi relatert forretningskontekst, slik at oppgavene kan fullføres korrekt og til rett tid.</span><span class="sxs-lookup"><span data-stu-id="92088-107">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="92088-108">Oppgavebehandling er en produktivitetsfunksjon i Dynamics 365 Commerce som lar overordnede og arbeidere opprette oppgavelister, behandle tildelingskriterier, spore oppgavestatus og integrere disse operasjonene mellom Commerce Back Office-programmer og POS-programmer (salgssted).</span><span class="sxs-lookup"><span data-stu-id="92088-108">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="92088-109">Personer i hovedkvarteret kan bruke oppgavebehandling til å opprette oppgavelister for detaljhandelsbutikker og til å spore status etter butikk eller arbeider.</span><span class="sxs-lookup"><span data-stu-id="92088-109">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="92088-110">De kan også opprette gjentakende oppgaver (for eksempel "Sjekkliste for lukking torsdags kveld").</span><span class="sxs-lookup"><span data-stu-id="92088-110">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="92088-111">Butikkledere kan bruke oppgavebehandling til å tilordne oppgaver til enkeltarbeidere, sende varsler om kommende oppgaver eller oppgaver som har forfalt, oppdatere oppgavestatus og opprette oppgaver med enkel bruk i POS-programmet.</span><span class="sxs-lookup"><span data-stu-id="92088-111">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="92088-112">Arbeidere kan deretter se varslinger, vise oppgavedetaljer og oppdatere oppgavestatus ved POS.</span><span class="sxs-lookup"><span data-stu-id="92088-112">Workers can then see notifications, view task details, and update task status at the POS.</span></span>

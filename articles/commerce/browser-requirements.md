# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Webleserkrav for Dynamics 365 Commerce-områder

[!include [banner](includes/banner.md)]

Dette emnet dekker nettleserkravene for områder for Microsoft Dynamics 365 Commerce.

Dynamics 365 Commerce-områder støttes best av følgende weblesere. Du finner den nyeste versjonen av hver nettleser ved å gå til programvareprodusentens nettsted.

- [Microsoft Edge](https://www.microsoft.com/edge) (opp til gjeldende versjon minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (opp til gjeldende versjon minus 1)
- [Google Chrome](https://www.google.com/chrome) (opp til gjeldende versjon minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (opp til gjeldende versjon minus 1)

> [!NOTE]
> Internet Explorer støttes ikke. Hvis du bruker Internet Explorer, kan det hende at noen funksjoner ikke fungerer, og brukerne kan ha en opplevelse som er under standard.

[!include [banner](../includes/banner.md)]

<span data-ttu-id="64c5e-104">Dette emnet beskriver hvordan du legger til felt i en Microsoft Excel-arbeidsbok, slik at du kan redigere detaljhandelstransaksjoner i Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="64c5e-104">This topic describes how to add fields to a Microsoft Excel workbook so that you can edit retail transactions in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="64c5e-105">Oversikt</span><span class="sxs-lookup"><span data-stu-id="64c5e-105">Overview</span></span>

<span data-ttu-id="64c5e-106">Når du genererer en Excel-fil slik at du kan redigere detaljhandelstransaksjoner, fylles filen med noen standardfelt.</span><span class="sxs-lookup"><span data-stu-id="64c5e-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="64c5e-107">Hvis et felt som må oppdateres, ikke vises som standard i den genererte Excel-filen, kan du legge det til.</span><span class="sxs-lookup"><span data-stu-id="64c5e-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="64c5e-108">Legge til felt i et regneark i en Excel-arbeidsbok</span><span class="sxs-lookup"><span data-stu-id="64c5e-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="64c5e-109">Hvis du vil legge til felt i en Excel-arbeidsbok, slik at du kan redigere detaljhandelstransaksjoner, følger du disse trinnene.</span><span class="sxs-lookup"><span data-stu-id="64c5e-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="64c5e-110">Last ned og åpne Excel-filen fra siden **Utdrag**, siden **Detaljhandelstransaksjoner** eller flisen **Transaksjonsvalideringsfeil** i arbeidsområdet **Finans for handelsbutikk**.</span><span class="sxs-lookup"><span data-stu-id="64c5e-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="64c5e-111">Velg **Utforming**.</span><span class="sxs-lookup"><span data-stu-id="64c5e-111">Select **Design**.</span></span>
1. <span data-ttu-id="64c5e-112">Velg blyantsymbolet for ønsket tabell, og finn og merk deretter feltet du vil legge til, i listen over tilgjengelige felt.</span><span class="sxs-lookup"><span data-stu-id="64c5e-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="64c5e-113">Velg **Legg til**, og velg deretter **Oppdater**.</span><span class="sxs-lookup"><span data-stu-id="64c5e-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="64c5e-114">Du kan endre rekkefølgen på felt.</span><span class="sxs-lookup"><span data-stu-id="64c5e-114">You can reorder fields.</span></span>
1. <span data-ttu-id="64c5e-115">Når oppdateringen er fullført, velger du **Oppdater** for å hente dataene for den nye kolonnen.</span><span class="sxs-lookup"><span data-stu-id="64c5e-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="64c5e-116">Det nye feltet og dataene for det skal nå være tilgjengelige for redigering i Excel.</span><span class="sxs-lookup"><span data-stu-id="64c5e-116">The new field and data for it should now be available for editing in Excel.</span></span>

## <a name="add-a-copyright-notice"></a><span data-ttu-id="a2742-109">Legge til en opphavsrettserklæring</span><span class="sxs-lookup"><span data-stu-id="a2742-109">Add a copyright notice</span></span>

<span data-ttu-id="a2742-110">Hvis du vil legge til en opphavsrettserklæring nederst på hver side som bruker en bestemt mal, følger du disse trinnene.</span><span class="sxs-lookup"><span data-stu-id="a2742-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="a2742-111">Gå til **Fragmenter**, og velg **Nytt sidefragment**.</span><span class="sxs-lookup"><span data-stu-id="a2742-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="a2742-112">I dialogboksen velger du **Bunntekst**-modulen, og deretter gir du navn til fragmentet.</span><span class="sxs-lookup"><span data-stu-id="a2742-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="a2742-113">Skriv for eksempel inn **Bunntekst-opphavsrettserklæring**.</span><span class="sxs-lookup"><span data-stu-id="a2742-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="a2742-114">Velg **OK**.</span><span class="sxs-lookup"><span data-stu-id="a2742-114">Select **OK**.</span></span>
1. <span data-ttu-id="a2742-115">I navigasjonsruten velger du ellipseknappen (**...**) ved siden av **Bunntekst**, og deretter velger du **Legg til modul**.</span><span class="sxs-lookup"><span data-stu-id="a2742-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="a2742-116">I dialogboksen velger du **Bunntekstkategori**, og deretter velger du **OK**.</span><span class="sxs-lookup"><span data-stu-id="a2742-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="a2742-117">I navigasjonsruten velger du ellipseknappen ved siden av **Bunntekstkategori**, og deretter velger du **Legg til modul**.</span><span class="sxs-lookup"><span data-stu-id="a2742-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="a2742-118">I dialogboksen velger du **Tekstblokk**, og deretter velger du **OK**.</span><span class="sxs-lookup"><span data-stu-id="a2742-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="a2742-119">I navigasjonsruten velger du **Tekstblokk**.</span><span class="sxs-lookup"><span data-stu-id="a2742-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="a2742-120">Legg til melding om opphavsrett i feltet **Avsnitt** i egenskapsruten til høyre.</span><span class="sxs-lookup"><span data-stu-id="a2742-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="a2742-121">Skriv for eksempel inn **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="a2742-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="a2742-122">Velg **Lagre**, velg **Sjekk inn**, og velg deretter **Publiser**.</span><span class="sxs-lookup"><span data-stu-id="a2742-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="a2742-123">Gå til **Maler**, velg malen, og velg deretter **Sjekk ut**.</span><span class="sxs-lookup"><span data-stu-id="a2742-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="a2742-124">Under **Sideoppsett** utvider du **Brødtekst**, og deretter utvider du **Standardside**.</span><span class="sxs-lookup"><span data-stu-id="a2742-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="a2742-125">Velg ellipseknappen ved siden av **Bunntekstspor**, og velg deretter **Legg til fragment**.</span><span class="sxs-lookup"><span data-stu-id="a2742-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="a2742-126">Velg fragmentet du opprettet tidligere, og velg deretter **Velg**.</span><span class="sxs-lookup"><span data-stu-id="a2742-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="a2742-127">Sjekk inn malen, og publiser den.</span><span class="sxs-lookup"><span data-stu-id="a2742-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="a2742-128">Bunnteksten som inneholder opphavsrettserklæringen, vises automatisk nederst på alle sider som bruker den valgte malen.</span><span class="sxs-lookup"><span data-stu-id="a2742-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>

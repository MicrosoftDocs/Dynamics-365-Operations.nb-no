# <a name="set-up-main-account-categories"></a>Definere hovedkontokategorier

[!include [task guide banner](../../includes/task-guide-banner.md)]

Hovedkontokategorier brukes til standardrapportene i finansiell rapportering og Power BI. Hovedkontokategorier som opprettes som standard kan gis nytt navn, men ikke slettes. Flere kontokategorier kan opprettes og brukes for rapportering og analyse. Denne oppgaven bruker demonstrasjonsfirmaet USMF.


## <a name="create-a-main-account-category"></a>Opprette en hovedkontokategori
1. Gå til Økonomimodul > Kontoplan > Kontoer > Hovedkontokategorier.
2. Klikk Ny.
3. Angi et unikt navn i feltet Hovedkontokategori.
4. Angi en beskrivelse for hovedkontokategorien i feltet Beskrivelse.
5. Velg typen hovedkonto som skal knyttes til kategorien i feltet Hovedkontotype.

## <a name="link-main-accounts-to-account-category"></a>Koble hovedkontoer til kontokategori
1. Klikk Koble hovedkontoer.
2. Velg hovedkontoene som skal tilordnes til hovedkontokategorien, i listen.
    * Når du tilordner hovedkontoer til en hovedkontokategori, samles saldoene for kontoene når denne kategorien brukes til økonomisk rapportering og analyse.  
3. Aktiver eller deaktiver alternativet Koblet for å velge hovedkontoene.
4. Klikk OK.
5. Klikk Ja.

2. <span data-ttu-id="c0929-106">Velg feltet du vil definere et filterkriterium for.</span><span class="sxs-lookup"><span data-stu-id="c0929-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="c0929-107">Skriv inn filterkriteriet for det valgte feltet.</span><span class="sxs-lookup"><span data-stu-id="c0929-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="c0929-108">Du kan filtrere på én enkelt verdi eller et område.</span><span class="sxs-lookup"><span data-stu-id="c0929-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="c0929-109">Når du definerer et område, kontroller at riktig syntaks brukes.</span><span class="sxs-lookup"><span data-stu-id="c0929-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="c0929-110">Verdiene må skilles med dobbelt punktum (..).</span><span class="sxs-lookup"><span data-stu-id="c0929-110">The values should be separated by a double period (..).</span></span>

8. <span data-ttu-id="c0929-115">Klikk kategorien Område.</span><span class="sxs-lookup"><span data-stu-id="c0929-115">Click the Range tab.</span></span>
9. <span data-ttu-id="c0929-116">Klikk OK for å kjøre spørringen.</span><span class="sxs-lookup"><span data-stu-id="c0929-116">Click OK to run the query.</span></span>

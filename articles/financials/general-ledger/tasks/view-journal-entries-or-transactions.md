# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="57dd5-103">Vise journaloppføringer eller transaksjoner</span><span class="sxs-lookup"><span data-stu-id="57dd5-103">View journal entries or transactions</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="57dd5-104">Denne prosedyren viser hvordan du bruker av bilagstransaksjonsforespørsler for å søke etter journaloppføringer eller transaksjoner.</span><span class="sxs-lookup"><span data-stu-id="57dd5-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="57dd5-105">Gå til Økonomimodul > Forespørsler og rapporter > Bilagstransaksjoner.</span><span class="sxs-lookup"><span data-stu-id="57dd5-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="57dd5-106">Velg feltet du vil definere et filterkriterium for.</span><span class="sxs-lookup"><span data-stu-id="57dd5-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="57dd5-107">Skriv inn filterkriteriet for det valgte feltet.</span><span class="sxs-lookup"><span data-stu-id="57dd5-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="57dd5-108">Du kan filtrere på én enkelt verdi eller et område.</span><span class="sxs-lookup"><span data-stu-id="57dd5-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="57dd5-109">Når du definerer et område, kontroller at riktig syntaks brukes.</span><span class="sxs-lookup"><span data-stu-id="57dd5-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="57dd5-110">Verdiene må skilles med dobbelt punktum (..).</span><span class="sxs-lookup"><span data-stu-id="57dd5-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="57dd5-111">Klikk kategorien Sammenkoblinger for å legge til flere tabeller som skal filtreres.</span><span class="sxs-lookup"><span data-stu-id="57dd5-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="57dd5-112">Velg Tabeller\Oppføring i økonomijournal.</span><span class="sxs-lookup"><span data-stu-id="57dd5-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="57dd5-113">Klikk Legg til tabellsammenkobling.</span><span class="sxs-lookup"><span data-stu-id="57dd5-113">Click Add table join.</span></span>
7. <span data-ttu-id="57dd5-114">Klikk Avbryt hvis du bestemmer deg for ikke å legge til en ekstra tabell.</span><span class="sxs-lookup"><span data-stu-id="57dd5-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="57dd5-115">Klikk kategorien Område.</span><span class="sxs-lookup"><span data-stu-id="57dd5-115">Click the Range tab.</span></span>
9. <span data-ttu-id="57dd5-116">Klikk OK for å kjøre spørringen.</span><span class="sxs-lookup"><span data-stu-id="57dd5-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="57dd5-117">Klikk Transaksjonsopprinnelse.</span><span class="sxs-lookup"><span data-stu-id="57dd5-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="57dd5-118">Ulike knapper i rutenettet kan brukes til å undersøke tilleggsinformasjon om den valgte posten for bilaget.</span><span class="sxs-lookup"><span data-stu-id="57dd5-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="57dd5-119">Enkelte knapper er kanskje ikke tilgjengelige, avhengig av type transaksjon og egenskaper for transaksjonen.</span><span class="sxs-lookup"><span data-stu-id="57dd5-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="57dd5-120">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="57dd5-120">Close the page.</span></span>
12. <span data-ttu-id="57dd5-121">Klikk Opprinnelig dokument.</span><span class="sxs-lookup"><span data-stu-id="57dd5-121">Click Original document.</span></span>
13. <span data-ttu-id="57dd5-122">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="57dd5-122">Close the page.</span></span>

3. <span data-ttu-id="b58e1-109">Velg en transaksjonen du vil undersøke.</span><span class="sxs-lookup"><span data-stu-id="b58e1-109">Select a transaction to research.</span></span>

    <span data-ttu-id="b58e1-111">Dialogboksen **Vis utligninger** åpnes og viser den valgte transaksjonen og alle dokumenter som er utlignet mot den.</span><span class="sxs-lookup"><span data-stu-id="b58e1-111">The **View settlements** dialog box opens and shows the selected transaction and all documents that are settled against it.</span></span> <span data-ttu-id="b58e1-112">Denne dialogboksen ligner på utligningsloggvisningen, men den inneholder alle relaterte dokumenter.</span><span class="sxs-lookup"><span data-stu-id="b58e1-112">This dialog box resembles the settlement history view, but it includes all related documents.</span></span>

5. <span data-ttu-id="b58e1-113">Du kan utføre forskjellige oppgaver i denne dialogboksen.</span><span class="sxs-lookup"><span data-stu-id="b58e1-113">You can perform various tasks from this dialog box.</span></span> <span data-ttu-id="b58e1-114">Velg ett eller flere bilag, og velg deretter én av følgende menyer:</span><span class="sxs-lookup"><span data-stu-id="b58e1-114">Select one or more vouchers, and then select one of the following menus:</span></span>

   - <span data-ttu-id="b58e1-117">**Eksporter** – eksporter de valgte bilagene til Microsoft Excel.</span><span class="sxs-lookup"><span data-stu-id="b58e1-117">**Export** – Export the selected vouchers to Microsoft Excel.</span></span>
   - <span data-ttu-id="b58e1-118">**Vis relaterte betalinger** – alle journaltransaksjoner for betaling som ble opprettet i betalingsjournalen som er knyttet til det valgte dokumentet, vises.</span><span class="sxs-lookup"><span data-stu-id="b58e1-118">**View related payments** – All the payment journal transactions that were created in the payment journal that is related to the selected document appear.</span></span> <span data-ttu-id="b58e1-119">I tillegg vises alle utligninger som er knyttet til disse betalingene.</span><span class="sxs-lookup"><span data-stu-id="b58e1-119">In addition, all the settlements that are related to those payments appear.</span></span> <span data-ttu-id="b58e1-120">Etiketten for denne menyen også endres til **Vis utligninger**.</span><span class="sxs-lookup"><span data-stu-id="b58e1-120">The label of this menu also changes to **View settlements**.</span></span> <span data-ttu-id="b58e1-121">Velg **Vis utligninger** for å vise bare transaksjonene som ble vist da du åpnet dialogboksen **Vis utligninger** for første gang.</span><span class="sxs-lookup"><span data-stu-id="b58e1-121">Select **View settlements** to show only the transactions that were shown when you first opened the  **View settlements** dialog box.</span></span>

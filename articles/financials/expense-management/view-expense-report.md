# <a name="view-an-expense-report"></a><span data-ttu-id="1c019-103">Vis en reiseregning</span><span class="sxs-lookup"><span data-stu-id="1c019-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1c019-104">Når du kjører en spørring for å vise transaksjoner i økonomimodulen eller Innkjøp og leverandører, kan spørringen gi for mye informasjon.</span><span class="sxs-lookup"><span data-stu-id="1c019-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="1c019-105">Det kan derfor hende at du vil begrense søket.</span><span class="sxs-lookup"><span data-stu-id="1c019-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="1c019-106">For å få informasjonen du trenger, kan du drille ned til en transaksjon for å se reiseregningen som den vises på.</span><span class="sxs-lookup"><span data-stu-id="1c019-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="1c019-107">For utgiftsrapporter kan du drille ned fra en transaksjon til den opprinnelige reiseregningen som er lagret i Reiseregning og utlegg.</span><span class="sxs-lookup"><span data-stu-id="1c019-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="1c019-108">I tillegg til det totale utgiftsbeløpet kan du vise de detaljerte transaksjonene, transaksjonene som er kvalifisert for merverdiavgiftsrefusjon (mva) og kvitteringene som er knyttet til utgiftene.</span><span class="sxs-lookup"><span data-stu-id="1c019-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="1c019-109">For å drille ned til den opprinnelige reiseregningen som en transaksjon vises på, velger du transaksjonen, og deretter **Originaldokumentet**.</span><span class="sxs-lookup"><span data-stu-id="1c019-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="1c019-110">Når du ser på individuelle transaksjoner, kan du også se tilleggsdata som er knyttet til en transaksjon.</span><span class="sxs-lookup"><span data-stu-id="1c019-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="1c019-111">Du kan for eksempel se en detaljert oversikt over hotellutgiftene, for eksempel Internett-tilgang, telefonsamtaler og room-service.</span><span class="sxs-lookup"><span data-stu-id="1c019-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>

# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="7b9be-103">Avhende et anleggsmiddel ved hjelp av en fritekstfaktura</span><span class="sxs-lookup"><span data-stu-id="7b9be-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="7b9be-104">Denne fremgangsmåten viser hvordan du kvitter deg med et anleggsmiddel ved hjelp av fritekstfakturaen.</span><span class="sxs-lookup"><span data-stu-id="7b9be-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="7b9be-105">Gå til Kunder > Fakturaer > Alle fritekstfakturaer.</span><span class="sxs-lookup"><span data-stu-id="7b9be-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="7b9be-106">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="7b9be-106">Click New.</span></span>
3. <span data-ttu-id="7b9be-107">Angi eller velg en verdi i Kundekonto-feltet.</span><span class="sxs-lookup"><span data-stu-id="7b9be-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="7b9be-108">Valider standard fakturadato, og rediger om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="7b9be-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="7b9be-109">Valider gjenværende standard hodefelt, for eksempel Valuta, og rediger om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="7b9be-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="7b9be-110">Angi en beskrivelse av fakturalinjen.</span><span class="sxs-lookup"><span data-stu-id="7b9be-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="7b9be-111">Angi eller velg hovedkontoen for fakturalinjen.</span><span class="sxs-lookup"><span data-stu-id="7b9be-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="7b9be-112">Valider standard mva-gruppe og mva-gruppe for varer.</span><span class="sxs-lookup"><span data-stu-id="7b9be-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="7b9be-113">Angi enhetsprisen eller salgsbeløpet til anleggsmidlet.</span><span class="sxs-lookup"><span data-stu-id="7b9be-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="7b9be-114">Klikk Linjedetaljer.</span><span class="sxs-lookup"><span data-stu-id="7b9be-114">Click Line details.</span></span>  
11. <span data-ttu-id="7b9be-115">Velg anleggsmiddelnummeret som skal selges.</span><span class="sxs-lookup"><span data-stu-id="7b9be-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="7b9be-116">Klikk Poster.</span><span class="sxs-lookup"><span data-stu-id="7b9be-116">Click Post.</span></span>

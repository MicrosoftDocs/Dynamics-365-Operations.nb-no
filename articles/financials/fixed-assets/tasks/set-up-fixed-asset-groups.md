[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="c38e6-104">Denne fremgangsmåten viser hvordan du oppretter en ny anleggsmiddelgruppe.</span><span class="sxs-lookup"><span data-stu-id="c38e6-104">This procedure shows how to create a new fixed asset group.</span></span> <span data-ttu-id="c38e6-105">Den bruker regnskapsførerrollen og demodataene for den juridiske enheten USMF.</span><span class="sxs-lookup"><span data-stu-id="c38e6-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="c38e6-106">Gå til Anleggsmidler > Oppsett > Anleggsmiddelgrupper.</span><span class="sxs-lookup"><span data-stu-id="c38e6-106">Go to Fixed assets > Setup > Fixed asset groups.</span></span>
2. <span data-ttu-id="c38e6-107">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="c38e6-107">Click New.</span></span>
3. <span data-ttu-id="c38e6-108">Skriv inn en verdi i feltet Anleggsmiddelgruppe.</span><span class="sxs-lookup"><span data-stu-id="c38e6-108">In the Fixed asset group field, type a value.</span></span>
4. <span data-ttu-id="c38e6-109">Skriv inn en verdi i Navn-feltet.</span><span class="sxs-lookup"><span data-stu-id="c38e6-109">In the Name field, type a value.</span></span>
    * <span data-ttu-id="c38e6-110">Automatisk nummerering av anleggsmidler og nummerseriekode i anleggsmiddelgruppen overstyrer innstillingene i parametere for anleggsmidler.</span><span class="sxs-lookup"><span data-stu-id="c38e6-110">Autonumber fixed assets and Number sequence code on the Fixed asset group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="c38e6-111">Du kan endre det her hvis anleggsmidlene i denne anleggsmiddelgruppen skal ha forskjellig nummerering fra andre grupper.</span><span class="sxs-lookup"><span data-stu-id="c38e6-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="c38e6-112">Klikk Tablåer.</span><span class="sxs-lookup"><span data-stu-id="c38e6-112">Click Books.</span></span>
6. <span data-ttu-id="c38e6-113">Angi eller velg en verdi i feltet Tablå.</span><span class="sxs-lookup"><span data-stu-id="c38e6-113">In the Book field, enter or select a value.</span></span>
    * <span data-ttu-id="c38e6-114">Feltet Beregn avskrivning er satt til Ja, så anleggsmiddeltablået inkluderes i avskrivningsforslag.</span><span class="sxs-lookup"><span data-stu-id="c38e6-114">The Calculate depreciation field is set to Yes, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="c38e6-115">Hvis Beregn avskrivning er satt til Nei, vil anleggsmidlet ikke automatisk avskrives.</span><span class="sxs-lookup"><span data-stu-id="c38e6-115">If Calculate depreciation is set to No, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="c38e6-116">Angi levetiden for anleggsmidlet i år.</span><span class="sxs-lookup"><span data-stu-id="c38e6-116">Set the Service life of the asset, in years.</span></span>

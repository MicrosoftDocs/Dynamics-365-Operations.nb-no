[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="dfa3f-104">Noen ganger kan ledige stillinger fylles av kandidater som allerede er ansatte i organisasjonen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="dfa3f-105">Denne prosedyren går gjennom trinnene for å ansette en eksisterende ansatt gjennom rekrutteringsprosessen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="dfa3f-106">I denne prosedyren er et rekrutteringsprosjekt allerede definert, og en eksisterende ansatt har allerede sendt en søknad for rekrutteringsprosjektet.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="dfa3f-107">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="dfa3f-108">Gå til Personale > Rekruttering > Søknader > Søknader.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>
2. <span data-ttu-id="dfa3f-109">Finn søknaden for den ansatte du vil ansette, i listen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-109">In the list, find application for the employee that you would like to hire.</span></span> <span data-ttu-id="dfa3f-110">Eksempel: 00002 John Emory</span><span class="sxs-lookup"><span data-stu-id="dfa3f-110">Example:  00002  John Emory</span></span>
3. <span data-ttu-id="dfa3f-111">Klikk Søknadsstatus.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-111">Click Application status.</span></span>
    * <span data-ttu-id="dfa3f-112">Søknadsstatus angir hvor en søknad er i rekrutteringsprosessen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="dfa3f-113">Hvert av disse trinnene er valgfrie.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-113">Each of these steps is optional.</span></span> <span data-ttu-id="dfa3f-114">Vanligvis vil en søknad flytte statusene i følgende rekkefølge: mottatt, bekreftet og intervjuet.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="dfa3f-115">Etter intervjuprosessen foretas en beslutning om ansettelse.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="dfa3f-116">Klikk Endre stilling.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-116">Click Change position.</span></span>
5. <span data-ttu-id="dfa3f-117">Velg stillingen som du skal ansette den ansatte i.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-117">Select the position that you are hiring the employee into.</span></span>
6. <span data-ttu-id="dfa3f-118">I feltet Ny startdato for tilordning angir du datoen da ansatt skal begynne å arbeide i den nye stillingen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>  
7. <span data-ttu-id="dfa3f-119">I feltet Sluttdato for tilordning angir du datoen da ansatt skal slutte å arbeide i den gjeldende stillingen.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-119">In the Assignment end date, enter the date that the employee will stop working in their current position.</span></span>
    * <span data-ttu-id="dfa3f-120">Startdatoen for den nye stillingen og sluttdatoen for den gamle stillingen kan overlappe.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="dfa3f-121">Dette kan skje når en person utfører oppgavene for begge stillinger i løpet av en overgangsperiode.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>  
8. <span data-ttu-id="dfa3f-122">Hvis du vil, kan du velge en årsakskode.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-122">Optionally, you can select a reason code.</span></span> <span data-ttu-id="dfa3f-123">Eksempel: Omorganisering</span><span class="sxs-lookup"><span data-stu-id="dfa3f-123">Example: Reorganization</span></span>
9. <span data-ttu-id="dfa3f-124">Klikk Endre stilling.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-124">Click Change position.</span></span>
    * <span data-ttu-id="dfa3f-125">Du kan også endre kompensasjon på dette tidspunktet.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-125">You may also change compensation at this time.</span></span> <span data-ttu-id="dfa3f-126">Hvis du ikke tilordner kompensasjon på dette tidspunktet, kan du endre den ved å gå til arbeiderskjemaet, velge kategorien Kompensasjon og velge Fast plan.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="dfa3f-127">Når du har valgt Endre stilling, oppdateres statusen for søknaden til Ansatt.</span><span class="sxs-lookup"><span data-stu-id="dfa3f-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>

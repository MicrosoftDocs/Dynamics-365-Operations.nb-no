# <a name="change-the-banners-or-logo-images-for-legal-entities"></a><span data-ttu-id="d2b29-103">Endre bannere eller logobilder for juridiske enheter</span><span class="sxs-lookup"><span data-stu-id="d2b29-103">Change the banners or logo images for legal entities</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d2b29-104">Følgende prosedyre viser hvordan systemansvarlige kan oppdatere banneret eller logobilde som vises for en juridisk enhet.</span><span class="sxs-lookup"><span data-stu-id="d2b29-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="d2b29-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="d2b29-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="d2b29-106">Gå til Organisasjonsstyring > Organisasjoner > Juridiske enheter.</span><span class="sxs-lookup"><span data-stu-id="d2b29-106">Go to Organization administration > Organizations > Legal entities.</span></span>

4. <span data-ttu-id="d2b29-111">Klikk Rediger</span><span class="sxs-lookup"><span data-stu-id="d2b29-111">Click Edit.</span></span>
5. <span data-ttu-id="d2b29-112">Velg Banner i feltet Firmabildetype for instrumentbord.</span><span class="sxs-lookup"><span data-stu-id="d2b29-112">In the Dashboard company image type field, select 'Banner'.</span></span>

    * <span data-ttu-id="d2b29-114">Tips!

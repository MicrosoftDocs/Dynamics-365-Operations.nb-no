# <a name="approve-application-inbox-records"></a>Godkjenne innboksposter for søknad

[!include [task guide banner](../../includes/task-guide-banner.md)]

Denne fremgangsmåten viser hvordan du går gjennom søknader som mottas gjennom selvbetjeningssidene for ansatte. I tillegg til å gå gjennom søknadene, kan du godkjenne søknaden i boksposter som du velger. Innboksposter for søknad representerer jobbsøknader som ble sendt til firmaet for vurdering. Etter at en post er godkjent, opprettes en søkerposten for personen som sendte inn søknaden. Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.


## <a name="review-application-inbox-record"></a>Se gjennom innbokspost for søknad
1. Gå til Personale > Rekruttering > Søknader > Søknadsinnboks.
2. Finn og velg ønsket post i listen.
3. Utvid seksjonen Adresser.
4. Vis delen Kontaktinformasjon.
5. Utvid delen Vedlegg.

## <a name="approve-application-inbox-record"></a>Godkjenn innbokspost for søknad
1. Merk den valgte raden i listen.
2. Merk deg verdien i Navn-feltet for senere bruk
3. Klikk Godkjenn.
4. Klikk OK.
5. Lukk siden.

## <a name="view-the-newly-created-application-record"></a>Vis den nyopprettede søknadsposten
1. Gå til Personale > Rekruttering > Søknader > Søknader.
2. Klikk koblingen i den valgte raden i listen.
3. Utvid delen Vedlegg.

# <a name="create-an-operating-unit"></a><span data-ttu-id="e464d-103">Opprette en driftsenhet</span><span class="sxs-lookup"><span data-stu-id="e464d-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e464d-104">En driftsenhet er en organisasjon som brukes til å fordele kontrollen over økonomiske ressurser og driftsprosesser i en virksomhet.</span><span class="sxs-lookup"><span data-stu-id="e464d-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="e464d-105">Personer i en driftsenhet har plikt til å gjøre mest mulig ut av knappe ressurser, forbedre prosesser og gjøre rede for egen ytelse.</span><span class="sxs-lookup"><span data-stu-id="e464d-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="e464d-106">Driftsenhetstypene omfatter kostsentre, forretningsenheter, avdelinger og verdistrømmer.</span><span class="sxs-lookup"><span data-stu-id="e464d-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="e464d-107">Bruk følgende fremgangsmåte til å opprette en driftsenhet.</span><span class="sxs-lookup"><span data-stu-id="e464d-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="e464d-108">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="e464d-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="e464d-109">Gå til **Navigasjonsrute > Moduler > Organisasjonsstyring > Organisasjoner > Driftsenheter**.</span><span class="sxs-lookup"><span data-stu-id="e464d-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="e464d-110">Klikk på **Ny** for å åpne nedtrekksdialogen.</span><span class="sxs-lookup"><span data-stu-id="e464d-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="e464d-111">Finn og velg ønsket post i listen.</span><span class="sxs-lookup"><span data-stu-id="e464d-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="e464d-112">Velg hvilken type driftsenhet du vil opprette.</span><span class="sxs-lookup"><span data-stu-id="e464d-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="e464d-113">Klikk koblingen i den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="e464d-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="e464d-114">Skriv inn en verdi i **Navn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="e464d-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="e464d-115">Utvid delen **Generelt** om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="e464d-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="e464d-116">Angi generelle opplysninger om driftsenheten, for eksempel ID-nummer, DUNS-nummer og sjef.</span><span class="sxs-lookup"><span data-stu-id="e464d-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="e464d-117">Utvid delen **Adresser** om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="e464d-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="e464d-118">Angi adresseinformasjon som for eksempel gatenavn og -nummer, postnummer og poststed.</span><span class="sxs-lookup"><span data-stu-id="e464d-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="e464d-119">Klikk på **Legg til** for å angi en ny adressepost, eller klikk Rediger for å endre en eksisterende adressepost.</span><span class="sxs-lookup"><span data-stu-id="e464d-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="e464d-120">Utvid delen **Kontaktinformasjon** om nødvendig.</span><span class="sxs-lookup"><span data-stu-id="e464d-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="e464d-121">Angi informasjon om kommunikasjonsmetoder, for eksempel e-postadresse, URL-adresser og telefonnumre.</span><span class="sxs-lookup"><span data-stu-id="e464d-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="e464d-122">Klikk Ny for å angi en ny kommunikasjonspost.</span><span class="sxs-lookup"><span data-stu-id="e464d-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="e464d-123">Hvis du vil endre en eksisterende kommunikasjonspost, klikker du på **Flere alternativer > Avansert**.</span><span class="sxs-lookup"><span data-stu-id="e464d-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="e464d-124">Du kan også endre **driftsenhetsnummeret** etter behov.</span><span class="sxs-lookup"><span data-stu-id="e464d-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="e464d-125">Vær oppmerksom på at dette nummeret er en unik idenitifier for den tilsvarende **Part**-posten, og kan ikke være det samme som en annen driftsenhet.</span><span class="sxs-lookup"><span data-stu-id="e464d-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="e464d-126">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="e464d-126">Select **Save**.</span></span>

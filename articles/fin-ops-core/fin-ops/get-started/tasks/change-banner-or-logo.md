[!include [banner](../../includes/banner.md)]

<span data-ttu-id="02500-104">Følgende prosedyre viser hvordan systemansvarlige kan oppdatere banneret eller logobilde som vises for en juridisk enhet.</span><span class="sxs-lookup"><span data-stu-id="02500-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="02500-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="02500-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="02500-106">Gå til **Navigasjonsrute > Moduler > Organisasjonsstyring > Organisasjoner > Juridiske enheter**.</span><span class="sxs-lookup"><span data-stu-id="02500-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>

4. <span data-ttu-id="02500-110">Velg **Endre**.</span><span class="sxs-lookup"><span data-stu-id="02500-110">Select **Change**.</span></span>
    
    <span data-ttu-id="02500-111">Den ideelle oppløsningen for et bannerbilde er 1920 x 281 piksler.</span><span class="sxs-lookup"><span data-stu-id="02500-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="02500-112">Ideell bredde for et logobilde er 350 piksler.</span><span class="sxs-lookup"><span data-stu-id="02500-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="02500-113">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="02500-113">Select **Save**.</span></span>
6. <span data-ttu-id="02500-114">Gå til **Navigasjonsrute > Moduler > Felles > Felles > Standard instrumentbord**.</span><span class="sxs-lookup"><span data-stu-id="02500-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="02500-115">Det nye banner- eller logobildet skal vises på instrumentbordet.</span><span class="sxs-lookup"><span data-stu-id="02500-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="02500-116">Du må kanskje endre firmaet til det du lastet opp banneret for, ved hjelp av firmavelgeren.</span><span class="sxs-lookup"><span data-stu-id="02500-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>

# <a name="change-the-banner-or-logo"></a><span data-ttu-id="dede7-103">Endre banner eller logo</span><span class="sxs-lookup"><span data-stu-id="dede7-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="dede7-104">Følgende prosedyre viser hvordan systemansvarlige kan oppdatere banneret eller logobilde som vises for en juridisk enhet.</span><span class="sxs-lookup"><span data-stu-id="dede7-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="dede7-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="dede7-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="dede7-106">Gå til **Navigasjonsrute > Moduler > Organisasjonsstyring > Organisasjoner > Juridiske enheter**.</span><span class="sxs-lookup"><span data-stu-id="dede7-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="dede7-107">Velg den juridiske enheten som du vil oppdatere banneret eller en logoen for, i listen til venstre.</span><span class="sxs-lookup"><span data-stu-id="dede7-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="dede7-108">Hvis det allerede er valgt, kan du gå til neste trinn.</span><span class="sxs-lookup"><span data-stu-id="dede7-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="dede7-109">Utvid fanen **Instrumentbordbilde**.</span><span class="sxs-lookup"><span data-stu-id="dede7-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="dede7-110">Velg **Endre**.</span><span class="sxs-lookup"><span data-stu-id="dede7-110">Select **Change**.</span></span>
    
    <span data-ttu-id="dede7-111">Den ideelle oppløsningen for et bannerbilde er 1920 x 281 piksler.</span><span class="sxs-lookup"><span data-stu-id="dede7-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="dede7-112">Ideell bredde for et logobilde er 350 piksler.</span><span class="sxs-lookup"><span data-stu-id="dede7-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="dede7-113">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="dede7-113">Select **Save**.</span></span>
6. <span data-ttu-id="dede7-114">Gå til **Navigasjonsrute > Moduler > Felles > Felles > Standard instrumentbord**.</span><span class="sxs-lookup"><span data-stu-id="dede7-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="dede7-115">Det nye banner- eller logobildet skal vises på instrumentbordet.</span><span class="sxs-lookup"><span data-stu-id="dede7-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="dede7-116">Du må kanskje endre firmaet til det du lastet opp banneret for, ved hjelp av firmavelgeren.</span><span class="sxs-lookup"><span data-stu-id="dede7-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e25bc-104">Følgende prosedyre viser hvordan systemansvarlige kan oppdatere banneret eller logobilde som vises for en juridisk enhet.</span><span class="sxs-lookup"><span data-stu-id="e25bc-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="e25bc-105">Demonstrasjonsdatafirmaet USMF brukes til å opprette denne fremgangsmåten.</span><span class="sxs-lookup"><span data-stu-id="e25bc-105">The demo data company used to create this procedure is USMF.</span></span>

4. <span data-ttu-id="e25bc-110">Velg **Endre**.</span><span class="sxs-lookup"><span data-stu-id="e25bc-110">Select **Change**.</span></span>
    
    <span data-ttu-id="e25bc-111">Den ideelle oppløsningen for et bannerbilde er 1920 x 281 piksler.</span><span class="sxs-lookup"><span data-stu-id="e25bc-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="e25bc-112">Ideell bredde for et logobilde er 350 piksler.</span><span class="sxs-lookup"><span data-stu-id="e25bc-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="e25bc-113">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="e25bc-113">Select **Save**.</span></span>

    <span data-ttu-id="e25bc-116">Du må kanskje endre firmaet til det du lastet opp banneret for, ved hjelp av firmavelgeren.</span><span class="sxs-lookup"><span data-stu-id="e25bc-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>

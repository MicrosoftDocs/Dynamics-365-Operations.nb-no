## <a name="operating-unit-to-msdyn_internalorganizations"></a>Driftsenhet til msdyn_internalorganizations

Denne malen synkroniserer data mellom Finance and Operations-apper og Common Data Service.

---|---|---|---

NAMEALIAS | > | msdyn_namealias | 
NAME | > | msdyn_name | 
PARTYNUMBER | > | msdyn_partynumber |

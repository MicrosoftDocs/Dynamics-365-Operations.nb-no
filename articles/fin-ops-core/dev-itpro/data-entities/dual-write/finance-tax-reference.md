# <a name="access-to-finance-and-tax-reference-data"></a>Tilgang til referansedata for finan og avgift

[!include [banner](../../includes/banner.md)]



Alle bedrifter arbeider med et grunnleggende sett med økonomiske data, for eksempel regnskapskalenderåret, valutaen som virksomheten utføres i, kontoene som pengene for å drive forretningen, kommer inn eller går ut av, mva-satser og remittering. Disse dataene finnes i Finance and Operations-apper. De vises imidlertid for Common Data Service slik at modelldrevne apper i Microsoft Dynamics 365 kan ha én enkelt kilde for økonomi- og avgiftsdata. På denne måten er data like på tvers av forretningsøkosystemet. 

Økonomi- og avgiftsdata integreres ved hjelp av følgende tilordninger:

+ [Integrert finans](ledger-mapping.md)
+ [Integrert original for avgift](tax-mapping.md)

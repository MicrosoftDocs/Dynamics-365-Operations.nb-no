# <a name="create-a-record-template-to-facilitate-data-entry"></a><span data-ttu-id="cb17e-103">Opprette en postmal for å forenkle dataregistrering</span><span class="sxs-lookup"><span data-stu-id="cb17e-103">Create a record template to facilitate data entry</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="cb17e-104">Dette emnet beskriver hvordan du oppretter en postmal slik at verdiene som brukes ofte, ikke trenger å angis eksplisitt for hver nye post.</span><span class="sxs-lookup"><span data-stu-id="cb17e-104">This topic demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="cb17e-105">I denne prosedyren skal du opprette en ny post for nye bærbare datamaskiner som skal legges til anleggsmidlene.</span><span class="sxs-lookup"><span data-stu-id="cb17e-105">In this procedure, you'll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="cb17e-106">Denne prosedyren bruker eksempelfirmaet USMF.</span><span class="sxs-lookup"><span data-stu-id="cb17e-106">This procedure uses the USMF sample company.</span></span>

1. <span data-ttu-id="cb17e-107">I navigasjonsruten går du til **Moduler > Anleggsmidler > Anleggsmidler > Anleggsmidler**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-107">In the navigation pane, go to **Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="cb17e-108">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-108">Select **New**.</span></span>
3. <span data-ttu-id="cb17e-109">Angi eller velg en verdi i feltet **Anleggsmiddelgruppe**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-109">In the **Fixed asset group** field, enter or select a value.</span></span>
4. <span data-ttu-id="cb17e-110">Skriv inn en verdi i **Navn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb17e-110">In the **Name** field, type a value.</span></span> <span data-ttu-id="cb17e-111">Angi for eksempel **firmakundeemne, bærbar datamaskin**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-111">For example, enter **Corporate lead laptop**.</span></span>  
5. <span data-ttu-id="cb17e-112">Skriv inn en verdi i **Søkenavn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb17e-112">In the **Search name** field, type a value.</span></span> <span data-ttu-id="cb17e-113">Angi for eksempel **bærbar datamaskin**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-113">For example, enter **laptop**.</span></span>  
6. <span data-ttu-id="cb17e-114">Vis delen **Teknisk informasjon**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-114">Expand the **Technical information** section.</span></span>
7. <span data-ttu-id="cb17e-115">Skriv inn verdier i feltene **Fabrikat**, **Modell** og **Modellår**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-115">In the **Make**, **Model**, and **Model year** fields, type values.</span></span>
8. <span data-ttu-id="cb17e-116">Velg **Alternativer** i handlingsruten.</span><span class="sxs-lookup"><span data-stu-id="cb17e-116">On the Action Pane, select **Options**.</span></span>
9. <span data-ttu-id="cb17e-117">Velg **Postinformasjon**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-117">Select **Record info**.</span></span>
10. <span data-ttu-id="cb17e-118">Velg **Brukermal**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-118">Select **User template**.</span></span>
11. <span data-ttu-id="cb17e-119">Skriv inn en verdi i **Navn**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb17e-119">In the **Name** field, type a value.</span></span>
12. <span data-ttu-id="cb17e-120">Skriv inn en verdi i **Beskrivelse**-feltet.</span><span class="sxs-lookup"><span data-stu-id="cb17e-120">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="cb17e-121">Velg **OK**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-121">Select **OK**.</span></span>
14. <span data-ttu-id="cb17e-122">Velg **Lukk**.</span><span class="sxs-lookup"><span data-stu-id="cb17e-122">Select **Close**.</span></span>

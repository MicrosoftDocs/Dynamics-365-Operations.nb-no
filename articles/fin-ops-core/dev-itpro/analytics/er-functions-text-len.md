# <a name="len-er-function"></a>LEN ER-funksjon

[!include [banner](../includes/banner.md)]

`LEN`-funksjonen returnerer antall tegn i den angitte strengen som en *Heltall*-verdi.

## <a name="syntax"></a>Syntaks

```vb
LEN (text)
```

## <a name="arguments"></a>Argumenter

`text`: *Streng*

En *streng*-verdi som angir teksten.

## <a name="return-values"></a>Returverdier

*Heltall*

Den resulterende numeriske verdien.

## <a name="example"></a>Eksempel

`LEN ("Sample")` returnerer **6**.

# <a name="lower-er-function"></a>LOWER ER-funksjonen

[!include [banner](../includes/banner.md)]

`LOWER`-funksjonen returnerer den angitte tekststrengen som en *Streng*-verdi etter at den er konvertert til sm√• bokstaver.

## <a name="syntax"></a>Syntaks

```vb
LOWER (text)
```

## <a name="arguments"></a>Argumenter

`text`: *Streng*

En *streng*-verdi som angir teksten.

## <a name="return-values"></a>Returverdier

*Streng*

Den resulterende tekstverdien.

## <a name="example"></a>Eksempel

`LOWER ("Sample")` returnerer **"sample"**.

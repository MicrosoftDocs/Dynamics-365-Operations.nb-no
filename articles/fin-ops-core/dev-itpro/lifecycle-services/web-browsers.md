# <a name="supported-web-browsers-for-lifecycle-services"></a>Støttede nettlesere for Lifecycle Services 

[!include[banner](../includes/banner.md)]

Brukere kan få tilgang til Microsoft Dynamics Lifecyle Services (LCS) ved å bruke de nyeste versjonene av disse ppulære nettleserne: 

- Microsoft Edge (anbefalt: [Chromium-basert Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Du finner den nyeste versjonen av hver nettleser ved å gå til programvareprodusentens nettsted. For optimal ytelse og opplevelse anbefaler vi at du bruker den nyeste versjonen av en nettleser, spesielt Microsoft Edge.

# <a name="payroll---gender-option-set"></a>Lønn – alternativ for kjønn angitt


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Denne artikkelen beskriver alternativetsettet for Kjønn for Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Alternativstett: alternativ for kjønn

Denne opplistingen gir alternativsettet for kjønn for den ansatte. Dette er tilgjengelig i mshr_hcmpersongender-alternativsettet.

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 | None | Det er ikke angitt noe kjønn. |
| 200000001 | Mann | Mann. |
| 200000002 | Kvinne | Kvinne. |
| 200000003 | Ikke-spesifikt | Valg for et ikke-spesifikt kjønn. |

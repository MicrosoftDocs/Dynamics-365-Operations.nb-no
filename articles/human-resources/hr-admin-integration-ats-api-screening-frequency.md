# <a name="screening-frequency"></a>Screeningfrekvens

Dette emnet beskriver alternativsettet for Screeningfrekvens for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmfrequencyunit

Denne opplistingen gir alternativsettet med verdier for screeningfrekvens. 

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 Bare én gang | Screeningen kreves bare én gang. |
| 200000001 Daglig | Screeningfrekvensen beregnes i dager. |
| 200000002 Ukentlig | Screeningfrekvensen beregnes i uker. |
| 200000003 Månedlig | Screeningfrekvensen beregnes i måneder. |
| 200000004 Årlig | Screeningfrekvensen beregnes i år. |

# <a name="screening-frequency-generate-from"></a>Screeningfrekvens genereres fra

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emnet beskriver screeningfrekvensen som genereres fra alternativsettet for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmfrequencygeneratefrom

Denne opplistingen gir alternativsettet med verdier for å bestemme startdatoen for beregningen for neste nødvendige screening.

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 Ikke valgt | Ingen verdi er valgt. |
| 200000001 Fullført dato | Beregningen utføres fra siste fullførte screeningdato. |
| 200000002 Dato påkrevd | Beregningen utføres fra siste påkrevde screeningdato. |

# <a name="create-employment-categories"></a>Opprett ansettelseskategorier

[!include [banner](includes/preview-feature.md)]

Definer ansettelseskategorier i Dynamics 365 Human Resources for å samsvare med arbeiderne.

1. I arbeidsområdet **Fordelsbehandling**, under **Oppsett**, velger du **Ansettelseskategorier**.

2. Velg **Ny**.

3. Angi verdier for **Arbeidertype**, **Ansettelseskategori** og **Beskrivelse**.

4. Velg **Lagre**.

# <a name="set-up-employment-types"></a>Definere ansettelsestyper

Definer ansettelsestyper i Dynamics 365 Human Resources for å samsvare med arbeiderne.

1. I arbeidsområdet **Personaladministrasjon**, under **Oppsett**, velger du **Ansettelsestyper**.

2. Velg **Ny**.

3. Angi verdier for de følgende feltene:

   | Felt | Beskrivelse |
   | --- | --- |
   | **Ansettelsestype** | Unik identifikasjon for ansettelsestypen. |
   | **Beskrivelse** | En beskrivelse av ansettelsestypen. |

4. Velg **Lagre**.

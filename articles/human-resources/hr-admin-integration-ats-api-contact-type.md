# <a name="contact-type"></a>Kontakttype

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I dette emnet beskrives alternativetsettet for Kontakttype for Dynamics 365 Human Resources.

Fysisk navn: mshr_logisticselectronicaddressmethodtype

Denne opplistingen gir alternativsettet med verdier for kandidatkontakttyper. 

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 | None | Ingen type er valgt. |
| 200000001 | Telefon | Telefonnummer. |
| 200000002 | E-postadresse | E-postadresse. |
| 200000003 | URL | URL-adresse for nettsted. |
| 200000004 | Teleks | Teleksnummer. |
| 200000005 | Faks | Faksnummer. |
| 200000006 | Facebook | Facebook-konto. Identifisert av Bruker-ID. |
| 200000007 | Twitter | Twitter-konto. Identifisert av @brukernavn. |
| 200000008 | LinkedIn | LinkedIn-konto. Identifisert av brukernavn. |

| Egenskap<br>**Fysisk navn**<br>**_Type_** | Bruk | beskrivelse |

| **Markedspris, lav terskel**<br>mshr_marketpricelowthreshold<br>*Desimal* | | En systemgenerert GUID-verdi som entydig identifiserer stillingen.  |

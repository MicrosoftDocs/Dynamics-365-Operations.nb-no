Dette emnet inneholder informasjon og en eksempelspørring for Jobbdetalj-enheten i Dynamics 365 Human Resources.

| Egenskap<br>**Fysisk navn**<br>**_Type_** | Bruk | beskrivelse |
| --- | --- | --- |
| **Jobb-ID**<br>mshr_jobid<br>*Streng* | Skrivebeskyttet<br>Obligatorisk | Unik ID for en jobb. |
| **Markedspris, lav terskel**<br>mshr_marketpricelowthreshold<br>*Desimal* | | En systemgenerert GUID-verdi som entydig identifiserer stillingen.  |

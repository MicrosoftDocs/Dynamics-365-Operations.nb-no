# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Fjernede eller avskrevne funksjoner i Dynamics 365 Human Resources

Denne artikkelen beskriver funksjoner som er fjernet eller som er planlagt for fjerning fra Microsoft Dynamics 365 Human Resources.

- En *fjernet* funksjon er ikke lenger tilgjengelig i produktet.
- En *avskrevet* funksjon er ikke i aktiv utvikling og kan bli fjernet i en fremtidig oppdatering.

Denne listen er ment å hjelpe deg med å vurdere disse fjerningene og avskrivningene for din egen planlegging.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Funksjoner som er fjernet eller avskrevet i Human Resources

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Årsak til avskrivning/fjerning** | Erstattet av oppretting av prosjekter i Finance and Operations-infrastrukturen. Hvis du vil ha mer informasjon om sammenslåingen av infrastrukturen, kan du se [Vanlige spørsmål om sammenslåingen av Dynamics 365 Human Resource-infrastrukturen](/hr-infrastructure-merge-faq). |
| **Erstattet med en annen funksjon?**   | Ja |
| **Berørte produktområder**         | Program |

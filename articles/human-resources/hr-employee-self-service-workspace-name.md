# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="300ab-103">Endre navn på arbeidsområde for selvbetjening for ansatte</span><span class="sxs-lookup"><span data-stu-id="300ab-103">Change Employee self service workspace name</span></span>

<span data-ttu-id="300ab-104">Hvis du har frivillige eller andre ikke-ansatte, kan det hende du vil endre navnet på arbeidsområdet **Ansattselvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="300ab-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="300ab-105">I stedet kan du endre dette arbeidsområdet til **Selvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="300ab-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="300ab-106">Hvis du endrer navnet på arbeidsområdet **Ansattselvbetjening**, endres også menyelementet som brukes internt av Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="300ab-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="300ab-107">Hvis du tidligere har brukt sikkerhetstilpassinger på menyelementet **HcmEmployeeSelfServiceWorkspace**, anbefaler vi at du bruker de samme endringene på **HcmSelfServiceWorkspace** for å opprettholde paritet.</span><span class="sxs-lookup"><span data-stu-id="300ab-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="300ab-108">I Human Resources velger du **Personaladministrasjon**, **Koblinger** og deretter **Personalparametere**.</span><span class="sxs-lookup"><span data-stu-id="300ab-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="300ab-109">Velg kategorien **Ansattselvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="300ab-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="300ab-110">Under **Visningsnavn** velger du **Selvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="300ab-110">Under **Display name**, select **Self service**.</span></span>

   ![Endre navn på arbeidsområde for selvbetjening til Selvbetjening](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="300ab-112">Velg **Lagre**.</span><span class="sxs-lookup"><span data-stu-id="300ab-112">Select **Save**.</span></span>

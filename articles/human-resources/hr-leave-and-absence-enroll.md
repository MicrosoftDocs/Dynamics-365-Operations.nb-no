# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="45299-103">Tilordne arbeidere til en permisjonsplan</span><span class="sxs-lookup"><span data-stu-id="45299-103">Assign workers to a leave plan</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

## <a name="assign-a-plan-to-a-group-of-employees"></a><span data-ttu-id="45299-105">Tilordne en plan til en gruppe av ansatte</span><span class="sxs-lookup"><span data-stu-id="45299-105">Assign a plan to a group of employees</span></span>

1. <span data-ttu-id="45299-106">På siden **Permisjon og fravær** velger du planen du vil tilordne.</span><span class="sxs-lookup"><span data-stu-id="45299-106">On the **Leave and absence** page, select the plan you want to assign.</span></span>

2. <span data-ttu-id="45299-107">Velg **Tilordne til ansatte**.</span><span class="sxs-lookup"><span data-stu-id="45299-107">Select **Assign to employees**.</span></span>

3. <span data-ttu-id="45299-108">Juster følgende innstillinger etter behov:</span><span class="sxs-lookup"><span data-stu-id="45299-108">Adjust the following settings as appropriate:</span></span>

   - <span data-ttu-id="45299-109">**Registrer på startdato for ansettelse**</span><span class="sxs-lookup"><span data-stu-id="45299-109">**Enroll on employment start date**</span></span>
   - <span data-ttu-id="45299-110">**Registreringsdato**</span><span class="sxs-lookup"><span data-stu-id="45299-110">**Enrollment date**</span></span>
   - <span data-ttu-id="45299-111">**Sluttdato**</span><span class="sxs-lookup"><span data-stu-id="45299-111">**End date**</span></span>
   - <span data-ttu-id="45299-112">**Basis for lag**</span><span class="sxs-lookup"><span data-stu-id="45299-112">**Tier basis**</span></span>

4. <span data-ttu-id="45299-113">Under **Arbeidere som skal registreres** velger du alle arbeiderne som skal tilordnes til planen.</span><span class="sxs-lookup"><span data-stu-id="45299-113">Under **Workers to enroll**, select all the works to assign to the plan.</span></span>

5. <span data-ttu-id="45299-114">Velg **Tilordne plan** nederst på siden.</span><span class="sxs-lookup"><span data-stu-id="45299-114">Select **Assign plan** at the bottom of the page.</span></span>

## <a name="assign-a-plan-to-an-employee"></a><span data-ttu-id="45299-115">Tilordne en plan til en ansatt</span><span class="sxs-lookup"><span data-stu-id="45299-115">Assign a plan to an employee</span></span>

1. <span data-ttu-id="45299-116">Velg **Permisjon** i posten for den ansatte.</span><span class="sxs-lookup"><span data-stu-id="45299-116">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="45299-117">Velg **Permisjon og fravær**.</span><span class="sxs-lookup"><span data-stu-id="45299-117">Select **Leave and absence**.</span></span>

3. <span data-ttu-id="45299-118">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="45299-118">Select **New**.</span></span>

4. <span data-ttu-id="45299-119">Angi en registreringsdato, og velg en permisjonsplan.</span><span class="sxs-lookup"><span data-stu-id="45299-119">Enter an enrollment date and select a leave plan.</span></span>

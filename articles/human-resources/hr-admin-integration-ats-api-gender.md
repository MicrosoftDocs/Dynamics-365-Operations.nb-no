# <a name="gender"></a>Kjønn

I dette emnet beskrives alternativet Kjønn for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmpersongender

Denne opplistingen gir alternativsettet for kjønn for kandidaten. Dette er tilgjengelig i mshr_hcmpersongender-alternativsettet.

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 | None | Det er ikke angitt noe kjønn. |
| 200000001 | Mann | Mann. |
| 200000002 | Kvinne | Kvinne. |
| 200000003 | Ikke-spesifikt | Valg for et ikke-spesifikt kjønn. |

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="15ba0-105">Suspendere permisjon og fravær for en ansatt</span><span class="sxs-lookup"><span data-stu-id="15ba0-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="15ba0-106">Velg **Permisjon** i posten for den ansatte.</span><span class="sxs-lookup"><span data-stu-id="15ba0-106">On the employee's record, select **Leave**.</span></span>

3. <span data-ttu-id="15ba0-108">Velg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="15ba0-108">Select **New**.</span></span>

4. <span data-ttu-id="15ba0-109">I dialogboksen **Suspender permisjonsavsetning** velger du **Permisjonstype** sammen med **Startdato** og **Sluttdato** for suspensjonen.</span><span class="sxs-lookup"><span data-stu-id="15ba0-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

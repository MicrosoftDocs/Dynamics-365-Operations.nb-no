# <a name="completion-status"></a>Fullføringsstatus

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I dette emnet beskrives alternativetsettet Fullføringsstatus for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmcompletionstatus

Denne opplistingen gir alternativsettet med statusverdier for kandidatscreeninger. 

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 | Ikke fullstendig | Kandidaten har ennå ikke fullført screeningen. |
| 200000001 | Bestått | Kandidaten har bestått screeningen. |
| 200000002 | Mislykket | Kandidaten har ikke bestått screeningen. |

# <a name="set-up-rounding-rules"></a>Definere avrundingsregler

[!include [banner](includes/preview-feature.md)]

Du kan definere avrundings regler i Dynamics 365 Human Resources for å imøtekomme organisasjonens behov.

1. I arbeidsområdet **Fordelsbehandling**, under **Oppsett**, velger du **Avrundingsregler**.

2. Velg **Ny**.

3. Angi verdier for **Avrund til nærmeste**, **Scenario** og **Avrundingsform**.

4. Velg **Lagre**.

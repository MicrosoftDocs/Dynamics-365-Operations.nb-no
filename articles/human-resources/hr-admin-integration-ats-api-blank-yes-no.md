# <a name="blank-yes-no"></a>Tomt Ja Nei

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

I dette emnet beskrives alternativetsettet Tomt Ja Nei for Dynamics 365 Human Resources.

Fysisk navn: mshr_hrmblankyesno

Denne opplistingen gir alternativsett for ja/nei-egenskaper som også tillater mellomrom.

| Verdi | Etikett | beskrivelse |
| --- | --- | --- |
| 200000000 | Blank | Ingen verdi er valgt. |
| 200000001 | Ja | Ja. |
| 200000002 | Nei | Nei |

# <a name="reporting-options-in-talent"></a>Rapporteringsalternativer i Talent

[!include [banner](includes/banner.md)]

**Miljødetaljer**

Dette problemet gjelder for alle miljøer.

**Symptom**

Kunden ønsker å tilpasse Microsoft Dynamics 365 for Talent-rapporter eller opprette nye rapporter.

**Utstede**

Brukeren kan ikke tilpasse de innebygde Microsoft Power BI-rapportene.

**Løsning**

- Core HR-dataene som flyter til Common Data Service for apper, kan rapporteres via PowerApps CDS-koblingen til Power BI Desktop. Legg merke til at Common Data Service for apper inneholder et delsett med Core HR-data. Hvis du vil ha mer informasjon om Power BI og instrumentbord, se [Opprette Power BI-rapporter og -instrumentbord med PowerApps Common Data Service](https://powerapps.microsoft.com/en-us/blog/cdsconnectortopowerbi).
- Elektronisk rapportering (ER) er tilgjengelig for noen av rapportene i Talent. Kundedrevne tilpasninger kan gjøres via ER-konfigurasjonsalternativene.
- Data kan eksporteres til Microsoft Excel eller Microsoft Word ved hjelp av ulike dataenheter som leveres av Talent via Microsoft Office-integreringen.

**Langsiktig løsning**

Flere Power BI-alternativer er tilgjengelige, og mer data og enheter vil være en del av Common Data Service for apper.

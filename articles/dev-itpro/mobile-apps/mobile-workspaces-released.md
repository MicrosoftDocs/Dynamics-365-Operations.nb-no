# <a name="mobile-workspaces"></a>Mobile arbeidsområder

[!include[banner](../includes/banner.md)]

Dette emnet viser de mobile arbeidsområdene som er tilgjengelige for bruk med mobilappen Microsoft Dynamics 365 for enhetlig drift.

> [!NOTE]
> Mobilappen var tidligere kalt *Microsoft Dynamics 365 for Finance and operations*.

| Mobilt arbeidsområde     | beskrivelse   | Tilgjengelighet   |
|----------------------|---------------|--------------|
|[Firmakatalog](company-directory-mobile-workspace.md)| Lar deg vise og kontakte andre ansatte i organisasjonen.| 2017. juni |    
|[Mitt team](manager-self-service-mobile-workspace.md)| Du kan vise direkte underordnede og underordnede og utvidede stab, samt gi ros til enkeltpersoner i rapporteringskjeden.|2017. juni |     
|[Fakturagodkjenning](invoice-approval-mobile-workspace.md)| Gir en oversikt over fakturaer som er tilordnet til deg gjennom arbeidsflytprosessen leverandørfakturahode.| 2017. juni   |
| [Reiseregning og utlegg](/dynamics365/unified-operations/financials/expense-management/expense-management-mobile-workspace) | Du kan registrere og laste opp en kvittering, slik at du kan knytte det til en reiseregningsrapport senere. Det mobile arbeidsområdet lar deg også raskt opprette en utgiftslinje ved hjelp av en tilknyttet kvittering. | 2017. april |
| [Godkjenning av bestilling](/dynamics365/unified-operations/supply-chain/procurement/purchase-order-mobile-workspace) | Vise og svare på bestillinger med handlinger som «godta» eller «avvis». | 2017. april |
| [Tidsoppføring for prosjekt](/dynamics365/unified-operations/financials/project-management/project-time-entry-mobile-workspace) | Du kan angi og lagre tid mot et prosjekt ved hjelp av mobilenheten. | 2017. mars |
| [Kostnadskontroll](/dynamics365/unified-operations/financials/cost-accounting/cost-controlling-mobile-workspace)     | Kostsenterledere kan se kostsenterresultater.                                                                                               |  2017. januar        |
| [Lagerbeholdning ](/dynamics365/unified-operations/supply-chain/inventory/inventory-on-hand-mobile-workspace)    | Få innsikt i reservert og tilgjengelig beholdning.                                                                                                    |   2017. januar       |
| [Salgsordrer](/dynamics365/unified-operations/supply-chain/sales-marketing/sales-orders-mobile-workspace)         | Du kan holde deg oppdatert om salgsordrene.                                                                                                                          |  2017. januar                  |
| [Leverandørsamarbeid](/dynamics365/unified-operations/supply-chain/procurement/vendor-collaboration-mobile-workspace) | Leverandører kan holde seg oppdatert om bestillinger som er sendt til dem for godkjenning. De kan også vise informasjon om nye og oppdaterte bestillinger og kontakter. |2017. januar    |

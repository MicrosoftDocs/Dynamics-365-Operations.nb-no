# <a name="configure-an-online-store"></a>Konfigurere en nettbutikk

Denne artikkelen inneholder koblinger til emner som hjelper deg med sentralt konfigurere og administrere og nettbutikk fra Microsoft Dynamics 365 for operasjoner.

Emnene som er oppført i tabellen nedenfor hjelper deg å konfigurere Dynamics 365 for operasjoner - Retail-komponenter og detaljhandel nettbutikken i Dynamics-365 for operasjoner-klienten.

## <a name="configure-an-online-store"></a>Konfigurere en nettbutikk
| Oppgave                                                | Detaljer                                                                                                                                                                                                                                                                                                                                                   | Emner                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Konfigurere Retail-komponenter.                        | Definere og vedlikeholde informasjon om detaljhandelsoperasjoner. Denne informasjonen inkluderer butikker, avgifter, produkter, gavekort, kampanjer og rabatter.                                                                                                                                                                                                          | [Konfigurere og vedlikeholde Retail](https://technet.microsoft.com/en-us/library/hh597201.aspx) (TechNet innhold for Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                          |
| Konfigurere en detaljhandelskanalen.    | Opprette en kategori detaljhandelskanalen til å definere en kategori struktur for produkter som du tilbyr gjennom en nettbutikk. Du definerer kategorihierarkiet, og tilordne produkter, attributt produktgrupper og attributtverdier til kategoriene. Deretter kan du tilordne kategorihierarkiet til en nettbutikk.                            | [Definere et hierarki for](https://technet.microsoft.com/en-us/library/hh580593.aspx) (TechNet innhold for Microsoft Dynamics AX 2012) [definerer attributter og attributtyper](https://technet.microsoft.com/en-us/library/hh227548.aspx) (TechNet innhold for Microsoft Dynamics AX 2012) [definert varegrupper attributtet](https://technet.microsoft.com/en-us/library/jj728713.aspx) (TechNet innhold for Microsoft Dynamics AX 2012) |
| Legge til nettbutikken organisasjonshierarkiet. | Før du kan tilordne produktsortimenter oppfylle ordrer for nettbutikken som du har opprettet eller generere rapporter som inneholder informasjon fra denne nettbutikken, må du tilordne butikken til ett eller flere organisasjonshierarkier. Du må minst tilordne nettbutikken til et organisasjonshierarki som inneholder produktsortimenter. | [Sette opp en nettbutikk](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet innhold for Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Legg til leveringsmåter til nettbutikken.          | Velg leveringsmetodene nettbutikken tilbyr.                                                                                                                                                                                                                                                                                                 | [Sette opp en nettbutikk](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet innhold for Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
| Tilordne attributter, og legge til metadata.                   | Velg alternativene som angir hvordan attributtene for hver kategori eller kanal produktet bør fungere i nettbutikken på Microsoft SharePoint-området.                                                                                                                                                                                              | [Sette opp en nettbutikk](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet innhold for Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |

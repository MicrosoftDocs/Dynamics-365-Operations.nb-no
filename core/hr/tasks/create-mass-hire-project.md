1. <span data-ttu-id="877fd-108">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="877fd-108">Click New.</span></span>
2. <span data-ttu-id="877fd-109">Skriv inn en verdi i feltet Masseansettelsesprosjekt.</span><span class="sxs-lookup"><span data-stu-id="877fd-109">In the Mass hire project field, type a value.</span></span>

6. <span data-ttu-id="877fd-113">Klikk Åpne prosjekt.</span><span class="sxs-lookup"><span data-stu-id="877fd-113">Click Open project.</span></span>
7. <span data-ttu-id="877fd-114">Klikk Ja.</span><span class="sxs-lookup"><span data-stu-id="877fd-114">Click Yes.</span></span>
8. <span data-ttu-id="877fd-115">Klikk Opprett stillinger.</span><span class="sxs-lookup"><span data-stu-id="877fd-115">Click Create positions.</span></span>
9. <span data-ttu-id="877fd-116">I Antall-feltet angir du antallet stillinger du vil opprette.</span><span class="sxs-lookup"><span data-stu-id="877fd-116">In the Quantity field, enter the number of positions that you want to create</span></span>

    * <span data-ttu-id="877fd-118">Sluttdatoen blir avslutningsdatoen for de nye arbeiderne.</span><span class="sxs-lookup"><span data-stu-id="877fd-118">The End date will be the Termination date for the new workers.</span></span>

    * <span data-ttu-id="877fd-125">Velg eventuelt avdelingen for de nye stillingene.</span><span class="sxs-lookup"><span data-stu-id="877fd-125">Optionally, select the Department for the new positions.</span></span>

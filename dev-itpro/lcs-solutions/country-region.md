# <a name="localization-and-regulatory-features"></a>Lokalisering og forskriftsmessige funksjoner

Microsoft Dynamics-356 for operasjoner inkluderer funksjonalitet for enkelte land-/ områdespesifikke bestemmelser og prosesser. Denne funksjonen er aktivert basert på den primære adressen for den aktive juridiske enheten. Dette emnet inneholder lister over ressurser som kan hjelpe deg å lære mer om land-/ områdespesifikk funksjonalitet.

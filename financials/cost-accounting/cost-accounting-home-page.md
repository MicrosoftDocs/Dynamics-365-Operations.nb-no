# <a name="cost-accounting-home-page"></a>Hjemmeside for kostnadsregnskap

Dette emnet inneholder en liste over hjelpeemner og andre ressurser som er tilgjengelige for kostnadsregnskap.

Kostnadsregnskapsmodulen kan du gjøre følgende:

-   Definer dimensjoner for kostnad-elementet, importere kostnader dimensjonsmedlemmer for elementet, og tilordne kostnader elementet dimensjoner.
-   Definer dimensjoner for kostnad-objektet, og Importer kostnader dimensjonsmedlemmer for objektet.
-   Definer eller importer statistiske dimensjoner.
-   Opprett maler for leverandør av statistisk måling.
-   Opprett kostnadsregnskapsfinanser.
-   Opprett kostnadskontrollenheter.
-   Behandle økonomimoduloppføringer.
-   Behandle budsjettoppføringer.
-   Opprett og bruk policyer for kostnadsatferd.
-   Spor kostnader.
-   Definer dimensjonshierarkier.
-   Lag setninger som kan vises i arbeidsområdet.
-   Opprett rapporter ved hjelp av Excel.

## <a name="cost-accounting-analysis-power-bi-content"></a>Kostregnskapsanalyse strøm BI-innhold
Kostregnskapsanalyse Microsoft Power BI-innholdet er ment for kontrollere kostnader eller andre som er ansvarlig for å utføre kostnadskontroll i en organisasjon. Det inkluderer hovedmål, for eksempel kostnader, størrelsen og kostnadssats av faktiske kostnader, budsjettkostnader og fleksible budsjettkostnader. Den bruker transaksjonsdata fra kostnadsregnskap i Microsoft Dynamics 365 for operasjoner, og gir en samlet visning av kostnadene for hele organisasjonen i en rapporteringsvaluta. Ledere kan filtrere dataene ved kostnad objekter til å utføre kostnadskontroll for sine organisasjonsenheter, selv om organisasjonen kan ha flere juridiske enheter.

-   [Kostregnskapsanalyse strøm BI-innhold](/dynamics365/operations/dev-itpro/analytics/cost-accounting-analysis-content-pack)
-   [Set up security for the Cost accounting analysis Power BI content](/dynamics365/operations/dev-itpro/analytics/setup-security-cost-accounting-content-pack)

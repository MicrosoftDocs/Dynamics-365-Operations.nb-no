# <a name="configure-cost-control-workspace-parameters"></a><span data-ttu-id="1117a-103">Konfigurere parametere for arbeidsområde for kostnadskontroll</span><span class="sxs-lookup"><span data-stu-id="1117a-103">Configure cost control workspace parameters</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="1117a-104">Bruk denne fremgangsmåten til å konfigurere kostnadskontrollarbeidsområdet slik at ledere på ulike nivåer i en organisasjon kan få innsikt i sin kostobjekter, for eksempel kostsentre og produktgrupper.</span><span class="sxs-lookup"><span data-stu-id="1117a-104">Use this procedure to configure the Cost control workspace so that managers at various levels in an organization can gain insight into their cost objects, such as cost centers and product groups.</span></span> <span data-ttu-id="1117a-105">USP2-demofirmaet ble brukt til å opprette denne registreringen.</span><span class="sxs-lookup"><span data-stu-id="1117a-105">The USP2 demo company was used to create this recording.</span></span>

1. <span data-ttu-id="1117a-106">Gå til Kostnadsregnskap > Oppsett > Konfigurasjon av arbeidsområde for kostnadskontroll.</span><span class="sxs-lookup"><span data-stu-id="1117a-106">Go to Cost accounting > Setup > Cost control workspace configuration.</span></span>
2. <span data-ttu-id="1117a-107">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="1117a-107">Click New.</span></span>
3. <span data-ttu-id="1117a-108">Skriv inn en verdi i Navn-feltet.</span><span class="sxs-lookup"><span data-stu-id="1117a-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="1117a-109">Skriv inn en verdi i feltet Beskrivelse.</span><span class="sxs-lookup"><span data-stu-id="1117a-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="1117a-110">Velg Ja i feltet Publisert.</span><span class="sxs-lookup"><span data-stu-id="1117a-110">Select Yes in the Published field.</span></span>
    * <span data-ttu-id="1117a-111">Hvis du setter dette alternativet til Ja, kan brukere som er tilordnet én av disse rollene se rapporten i kostnadskontrollarbeidsområdet: kostnadsregnskapsleder, regnskapsfører, regnskapsførerfunsksjonær eller kostnadsobjektkontroller.</span><span class="sxs-lookup"><span data-stu-id="1117a-111">If you set this option to Yes, users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, cost accountant clerk, or cost object controller.</span></span> <span data-ttu-id="1117a-112">Hvis du setter dette alternativet til Nei, kan bare brukere som er tilordnet én av disse rollene se rapporten i kostnadskontrollarbeidsområdet: kostnadsregnskapsleder, regnskapsfører, eller regnskapsførerfunsksjonær.</span><span class="sxs-lookup"><span data-stu-id="1117a-112">If you set this option to No, only users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, or cost accountant clerk.</span></span>  
6. <span data-ttu-id="1117a-113">Utvid delen Datafiltrering.</span><span class="sxs-lookup"><span data-stu-id="1117a-113">Expand the Data filtering section.</span></span>
7. <span data-ttu-id="1117a-114">Angi eller velg en verdi i feltet Kostnadskontrollenhet</span><span class="sxs-lookup"><span data-stu-id="1117a-114">In the Cost control unit field, enter or select a value.</span></span>
8. <span data-ttu-id="1117a-115">Angi eller velg en verdi i feltet Opprinnelig budsjettversjon.</span><span class="sxs-lookup"><span data-stu-id="1117a-115">In the Budget original version field, enter or select a value.</span></span>
9. <span data-ttu-id="1117a-116">Angi eller velg en verdi i feltet Dimensjonshierarki for kostnadselement.</span><span class="sxs-lookup"><span data-stu-id="1117a-116">In the Cost element dimension hierarchy field, enter or select a value.</span></span>
10. <span data-ttu-id="1117a-117">Angi eller velg en verdi i feltet Dimensjonshierarki for kostnadsobjekt.</span><span class="sxs-lookup"><span data-stu-id="1117a-117">In the Cost object dimension hierarchy field, enter or select a value.</span></span>
11. <span data-ttu-id="1117a-118">Utvid delen Tilordne beregningsposter.</span><span class="sxs-lookup"><span data-stu-id="1117a-118">Expand the Assign calculation records section.</span></span>
12. <span data-ttu-id="1117a-119">Klikk Ny.</span><span class="sxs-lookup"><span data-stu-id="1117a-119">Click New.</span></span>
13. <span data-ttu-id="1117a-120">Merk den valgte raden i listen.</span><span class="sxs-lookup"><span data-stu-id="1117a-120">In the list, mark the selected row.</span></span>
14. <span data-ttu-id="1117a-121">Angi eller velg en verdi i feltet Regnskapskalenderperiode.</span><span class="sxs-lookup"><span data-stu-id="1117a-121">In the Fiscal calendar period field, enter or select a value.</span></span>
15. <span data-ttu-id="1117a-122">Angi eller velg en verdi i feltet Faktisk versjon.</span><span class="sxs-lookup"><span data-stu-id="1117a-122">In the Actual version field, enter or select a value.</span></span>
16. <span data-ttu-id="1117a-123">Utvid delen Regnskapsperioder per kolonne.</span><span class="sxs-lookup"><span data-stu-id="1117a-123">Expand the Fiscal periods per column section.</span></span>
17. <span data-ttu-id="1117a-124">Velg Ja i feltet Gjeldende periode.</span><span class="sxs-lookup"><span data-stu-id="1117a-124">Select Yes in the Current period field.</span></span>
18. <span data-ttu-id="1117a-125">Utvid delen Kolonner som skal vises for kostnader.</span><span class="sxs-lookup"><span data-stu-id="1117a-125">Expand the Columns to display for costs section.</span></span>
19. <span data-ttu-id="1117a-126">Velg Ja i feltet Fast kostnad.</span><span class="sxs-lookup"><span data-stu-id="1117a-126">Select Yes in the Fixed cost field.</span></span>
20. <span data-ttu-id="1117a-127">Velg Ja i feltet Variabel kostnad.</span><span class="sxs-lookup"><span data-stu-id="1117a-127">Select Yes in the Variable cost field.</span></span>
21. <span data-ttu-id="1117a-128">Velg Ja i feltet Total kostnad.</span><span class="sxs-lookup"><span data-stu-id="1117a-128">Select Yes in the Total cost field.</span></span>
22. <span data-ttu-id="1117a-129">Klikk Lagre.</span><span class="sxs-lookup"><span data-stu-id="1117a-129">Click Save.</span></span>
23. <span data-ttu-id="1117a-130">Lukk siden.</span><span class="sxs-lookup"><span data-stu-id="1117a-130">Close the page.</span></span>
24. <span data-ttu-id="1117a-131">Gå til Kostnadsregnskap > Arbeidsområder > Kostnadskontroll.</span><span class="sxs-lookup"><span data-stu-id="1117a-131">Go to Cost accounting > Workspaces > Cost control.</span></span>
25. <span data-ttu-id="1117a-132">Velg en oppgave for å se faste, variable, totale og uklassifiserte kostnader for de valgte regnskapsperiodene.</span><span class="sxs-lookup"><span data-stu-id="1117a-132">Select a statement to see fixed, variable, total, and unclassified costs for the selected fiscal periods.</span></span>
26. <span data-ttu-id="1117a-133">Angi eller velg en verdi i feltet Regnskapskalenderperiode.</span><span class="sxs-lookup"><span data-stu-id="1117a-133">In the Fiscal calendar period field, enter or select a value.</span></span>
27. <span data-ttu-id="1117a-134">Angi eller velg en verdi i feltet Dimensjonshierarkinode for kostnadsobjekt.</span><span class="sxs-lookup"><span data-stu-id="1117a-134">In the Cost object dimension hierarchy node field, enter or select a value.</span></span>
    * <span data-ttu-id="1117a-135">Når du har valgt et dimensjonshierarki for kostnadsobjektet, utvid dimensjonshierarkiet for kostnadselementer for å vise de ønskede kostnadsverdier.</span><span class="sxs-lookup"><span data-stu-id="1117a-135">After you've selected a Cost object dimension hierarchy, expand the Cost element dimension hierarchy to see the desired cost values.</span></span> <span data-ttu-id="1117a-136">Du kan for eksempel utvide hierarkiet til indirekte produksjonskostnader hvis du vil vise verdien.</span><span class="sxs-lookup"><span data-stu-id="1117a-136">For example, you can expand the hierarchy to Manufacturing overhead to see the value.</span></span>

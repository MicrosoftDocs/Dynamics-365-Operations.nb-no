# <a name="create-and-submit-vat-report"></a>Opprette og sende inn mva-rapport

[!include[task guide banner](../../includes/task-guide-banner.md)]

Denne prosedyren hjelper deg med å opprette og sende inn mva-rapporten til tyske myndigheter.

Denne fremgangsmåten ble opprettet med demonstrasjonsdatafirmaet DEMF. 

Denne funksjonen er tilgjengelig for juridiske enheter med primæradresse i Tyskland.



1. Gå til Avgift > Deklareringer > Merverdiavgift > Rapporter merverdiavgift for utligningsperioden.
2. Angi eller velg en verdi i feltet Utligningsperiode.
3. Angi en dato i Fra dato-feltet.
4. Klikk OK.
5. Velg Ja i feltet Opprett elektronisk avgiftsdokument.
6. Klikk OK.
